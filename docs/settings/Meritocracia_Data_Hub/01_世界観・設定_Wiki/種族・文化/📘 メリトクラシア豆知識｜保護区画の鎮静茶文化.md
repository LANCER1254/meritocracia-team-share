📘 メリトクラシア豆知識｜保護区画の鎮静茶文化

対象: 審問庁・ZG保護係 / Scene29 応接室演出補助
用途: 本文小物・キャラ会話・読者の現実接続ギミック

1. 正式設定

審問庁の保護区画では、
過酷な聴取・奴隷保護・トラウマ証言の直後に、鎮静用ハーブティーを出す慣例がある。

これは優しさだけではなく、国家実務として標準化された精神安定処置。

担当は主に

女性保護係
医療補助官
VA観測補佐
ZG福祉記録官

のいずれか。

制度は冷たいが、心を壊したまま証言を続けさせない合理性として運用されている。

2. Scene29推奨配合（正式採用）
🌿 カモミール＋ベルガモット

アイリス専用演出として最適。

効果
緊張緩和
呼吸安定
嗚咽後の震えを落ち着かせる
夜に悪夢を見にくくする
聴取継続を可能にする
香り演出
カモミール = 花の甘さ
ベルガモット = 柑橘と紅茶香
名前「アールグレイ」と自然接続

女性保護係が

「あなたの名前に少し似せています」
と出すと感情的に強い。

3. ガチ現実うんちく（仕掛け）
アピゲニン

カモミールに含まれる天然フラボノイド。

現実でも
GABA系へ穏やかに作用し、不安・緊張を和らげる
とされる鎮静成分。

本作ではこれを
薬草班の基礎常識
として扱うとリアリティが高い。

例セリフ:

「カモミールのアピゲニンは、張り詰めた神経を少しだけ緩めます」

読者が後で調べると
“本当にある”
となる仕掛け。

4. メリトクラシア文化への落とし込み

能力主義国家であっても、
証言者や被害者を壊しては情報精度が落ちる。

そのため国家機関では
精神安定＝情報保全
の意味も持つ。

つまりこのお茶は

優しさ
実務
証言精度維持
国家合理性

を同時に満たす文化。

かなりメリトクラシアらしい。

5. 本文での使い方

本文では説明しすぎず、女性保護係の一言だけで十分。

「カモミールにベルガモットを少し。緊張をほどく配合です」

この一文だけで、
スタッフ側は裏でこの設定を共有しておけばOKです。
