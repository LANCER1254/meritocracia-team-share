# Claude差分改修プロンプト｜iris.html Day/Night切替追加

今ある `iris.html` の完成版をベースに、
ゼロから作り直さず **差分改修** してください。

## 🎯 目的

現在は Day Mode（紅茶ラベルUI）のみ完成している。
これを壊さず、同一HTML内に

> 🌙 Nightfall Mode（夜化因子UI）

を追加したい。

---

## 必須改修

### 1) モード切替ボタン追加

ヘッダー上部に以下を追加。

```html id="w8p7ca"
<button id="toggleModeBtn">☀ Day / 🌙 Night</button>
```

クリックで Day / Night 切替。

---

### 2) CSSテーマ切替

既存の Day CSS は維持。

Night追加時のみ以下へ反転。

* 背景：深紺 `#0B1020`
* 枠線：銀 `#C0C0C0`
* テキスト：月白
* hover：紫発光
* 湯気 → 黒粒子
* 金装飾 → 銀鎖・月蝕

---

### 3) hidden notes の中身切替

Night時だけタイトル変更。

> 深層心理 → 夜化因子記録

内容：

* 古代エルフ分岐
* 夜化因子隔世発現
* 家庭差別
* 戦争価値
* 壊れる直前人格

---

### 4) JS追加

class切替で実装。

```javascript id="1qf1wn"
document.body.classList.toggle('night-mode');
```

---

## 🎯 最終条件

今の完成度を崩さず、

> 昼は高級ティーメニュー
> 夜は禁書館カルテ

に反転するUIへアップデートしてください。
