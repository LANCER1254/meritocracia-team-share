📜 本編執筆済みセクション 統合議事録（ep01-ep05）
1. 物語のフェーズ：Phase 1「選別と静圧」

本編5話までの執筆により、以下の世界観構造が描写として確定しました。

    「合格＝通過」の徹底: 家族や周囲が合格を「祝祭」ではなく、当然の「手続き」として扱う空気感（ep04, 05）。

    感情の抑制: 主人公ジェイドのモノローグを最小限にし、淡々とした行動で「異常なまでの適応」を表現。

2. 各話の進捗と確定描写
ep01：はじまりの階段

    確定事項: 試験当日の朝の空気感。

    描写の核心: 石畳の音、白い息、検問の無機質さ。「黒衣」の存在を「見るだけの監視者」として初登場させた。

ep02：ライナルト登場

    確定事項: ライナルトとの接触、および《除番》の視覚的提示。

    描写の核心:

        対比: 華やかなグローリアテストへの導線と、奥にある「灰色の扉（除番）」の対比。

        ライナルト: 単なる悪役ではなく、淡々と「受かるといいな」と言える、制度への自負を持つ少年として描写。

        フラン: 灰色の紋章を背負い、名前のない「数字」として扉へ消える姿を確定。

ep03：試験本番

    確定事項: M-GSLに基づいた「構文設計」としての試験内容。

    描写の核心:

        設問: 「局所損傷の修復（再生）」における死亡事故防止。

        心音: ジェイドの心音が「警告」ではなく、設計上の「揺れ」として機能。

        演出: 魔法の火花ではなく「ペン音と紙の音」が支配する静寂。

ep04：七日間の静圧

    確定事項: 試験終了から合格発表までの「空白の7日間」。

    描写の核心:

        家庭の空気: 父の「終わったなら、次だな」という一言。制度が生活の末端まで浸透している様子。

        社会の反応: 不合格者が出ても、制度を疑わず「仕方ない」とする諦念。

ep05：掲示板の向こう側

    確定事項: 合格発表とアイリスの初登場。

    描写の核心:

        アイリスとカール: 貴族カールによる「商品管理」という言葉を用いたアイリスへの圧迫。

        ジェイドの行動原理: 自分が合格したかどうか（資格があるか）を確認する前に、アイリスの騒動に対して「胸が波打つ」感覚に従って動く。

3. 設定の定着（システム面）

    審問庁直轄・第一試験棟: 1階の受付、2階の扇形講義室、地下の監視構造が描写を通じて空間として確定。

    監視体制: 「監視水晶」と「動かない黒衣」による、常に「見られている」という圧力の定着。

4. 今後の執筆に向けた「残。課題」

    ジェイドの魔力量評価: 試験官による「中等、安定値……異常なし」という評価の裏にある違和感の回収。

    倫理試問の内実: ep03で示唆された、ジェイドの「核」に触れる試問の詳細描写。
