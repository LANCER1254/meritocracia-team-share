16. 救護区画（Scene21）

市場の火勢が抑えられた後、
FOと医療班が救護区画を展開する。

ジェイドはカーネルを抱えたまま動けない
白ローブ医療班到着
生存者優先処置

「……脈、なし。死亡確認」

ここで第三者による死亡宣告を入れることで、
カーネルの死が物語上の“現実”として固定される。

「まだ温かいのに助からない」
で統一。
