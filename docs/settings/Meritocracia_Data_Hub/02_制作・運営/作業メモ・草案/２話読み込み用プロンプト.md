【作品情報】
作品名：メリトクラシア -Reboot-
話数：第2話
形式：一人称・ノベルゲーム前提（地の文多め）
視点：主人公ジェイド固定（一人称）
ジャンル：異世界／実力主義／社会構造

【第1話までの状態】
・主人公は「選んだ」わけではなく、試験と制度に巻き込まれた状態
・思想・誓い・正義感は未確定
・賢さや達観は見せていない
・被害者性はあるが、被害を語っていない
・プロローグの重さ（テロ事件）はまだ消えていない
・読者は「何が起きているか完全には分かっていない」

【第2話の目的】
・第1話の余韻を壊さず、世界に「馴染まされていく」感覚を描く
・主人公が前向きになる展開は禁止
・主人公が答えを出す展開は禁止
・「違和感」「距離感」「慣れ」を中心に進める

【禁止事項（重要）】
・主人公が自分の状況を分析・解説する描写
・制度・国家・階級の説明
・主人公の決意、誓い、覚悟の明文化
・「俺はこう思った」「だから〜すべきだ」という思想表明
・主人公を有能・冷静・正義的に見せる演出
・説教臭い地の文

【必須要素】
・一人称視点の身体感覚（音・視線・距離）
・「何かおかしいが、言葉にできない」場面を複数入れる
・周囲の人物は“正しいことを言っているように見える”
・主人公は納得も反論もしない
・読者に説明しないまま場面を進める

【第2話の構造（推奨）】
1. 試験後の待機／移動／説明を受ける場面
2. 主人公が「自分だけ分かっていない」感覚を覚える
3. 他の受験者たちの自然な振る舞いとの距離感
4. 小さな出来事（会話／指示／視線）で違和感が増す
5. はっきりしないまま終わるラスト

【トーン指定】
・静か
・淡々
・感情を言語化しすぎない
・余白多め
・ノベルゲームで読み進めやすい短文構成

【出力条件】
・本文のみを出力する
・話数タイトルは不要（後で差し替える）
・800〜1200文字程度
・一人称「俺」で統一
