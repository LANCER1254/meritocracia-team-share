【最高機密：国家指導者所見】

【機密区分：最上位／国家機密指定】
【発言者：ロータス＝ヴァル＝ロータス（公爵／国家代表）】
【記録官：ヴィオラ＝スティルネン】

……彼は、"可能性"だ。

火種を持った少年。
それを灯せば、大きな炎になる。
放っておけば、静かに消えるかもしれない。

どちらが正しいか——もう誰にも分からない。

私たちはいつも、燃やしすぎるか、守りすぎるかのどちらかだ。
ちょうどいい加減など、存在しない。

彼が見せた"光"を、私は奪いたくない。
だが、その光が誰かを焼く日が来るなら——
それを止めるのは、私の役目だ。

国家の名において、私は見守る。
秩序の名において、私は記録する。

彼の未来は、まだ"観察中"である。

――ロータス＝ヴァル=ロータス

観察されていたという、事実。

そしてこの文を読み終えた君もまた、
いま、記録の中に居る。
