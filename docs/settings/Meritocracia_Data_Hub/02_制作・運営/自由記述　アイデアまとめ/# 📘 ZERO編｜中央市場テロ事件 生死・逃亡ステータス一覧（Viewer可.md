## 2. Viewerで視える化する表示ルール

### 🎨 カード色分け

* **死亡**: 赤系
* **逮捕**: 金系
* **逃亡 / 行方不明**: 紫系
* **生存（民間）**: 青系
* **黒幕候補 / 要監視**: 黒系アクセント

### 🧩 推奨UIカラム

* 識別タグ
* 名前
* 役割
* 現在状態
* 公開ステータス
* 再登場フック
