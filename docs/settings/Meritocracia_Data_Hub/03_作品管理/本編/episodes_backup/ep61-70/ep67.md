編集
前書き
塔の第二層で、静寂と囁きがジェイドたちを包み込みます。

精神干渉によって見せられたのは、“自分ではない誰か”の記憶。

その視線の先に立つ、赤い瞳の少女は――。

指輪と紋章が呼応する中、ソルヴェンの口から出た“資格者”という言葉が、さらに疑念を深めます。

本文
塔の第二層に足を踏み入れた瞬間、空気が一変した。

耳を塞ぎたくなるほどの静寂。

音が消えたはずなのに、微かな耳鳴りが頭蓋の奥を叩き続ける。



ジェイドは反射的に周囲を見渡す。

魔法灯の光は淡く揺れ、壁の陰影が人影のように形を変えていた。

後ろを振り返ると、キールが肩を震わせている。

「……やべぇ、何か…声が」

その顔色は蒼白で、瞳の焦点が揺らいでいた。



フィーネも額に手を当て、唇をかすかに震わせている。

「……聞こえる……でも、言葉にならない」

その耳はぴくぴくと動き、まるで何かに呼ばれているかのようだった。



ジェイドは一歩前へ出ようとしたが、膝の力が抜ける。

同時に、誰かの声が脳裏をかすめた。

──おまえは……誰だ……

低く、遠く、けれど確かに自分に向けられた声。



次の瞬間、視界が揺れ、色が滲んだ。

現実の床が崩れ落ち、黒い水面の上に立っている自分を見下ろす。

その奥、赤い光が二つ、闇を切り裂くように輝いた。



「……っ！」

ジェイドは息を呑み、意識を戻そうと瞬きを繰り返す。

だが耳鳴りは強まり、声は重なっていく。

──来るな……来るな……それでも……来い……



ソルヴェンだけが平然と歩を進め、背中を一度も振り返らなかった。



──気づけば、ジェイドは誰かの視界を借りていた。



重たい黒衣の袖口が揺れ、冷たい雨が頬を打つ感覚が、まるで自分のもののように生々しい。

足元には、瓦礫と炎に覆われた街。

遠くで、悲鳴が途切れる音がした。



（……これは、俺じゃない）

そう確信するのに、視界から伝わる感覚は否応なく自分の体に染み込んでくる。

なぜこの光景が目に浮かぶ？

見知らぬ街、なのに……胸の奥が、焼けるように痛い。



視線が勝手に動き、人々の群れを見下ろす。

その瞳は深く濁り、怒りと絶望で燃えていた。



広場の中央、縛られた異種族たちが列を成している。

尖った耳のエルフ、角を折られた魔族、翼をもがれた堕天使……

彼らは冷たい雨に打たれながら、必死に顔を上げていた。



「裏切り者どもに、相応しい結末を──」

声が響くたび、群衆が歓声を上げる。

処刑台に立つ者の視界は、憤怒に震えていた。



その群衆の奥に――赤い瞳の少女が立っていた。

濡れた金髪、幼さを残す顔立ち。

（……デイジア……？）

似ている。しかし、その口元の笑みは冷たく、今の彼女が決して見せないものだった。



黒衣の腕が振り上げられ、視界が真紅に染まる。

「やめろ……！」

叫んだつもりが声にならず、世界が崩れ落ちる。



現実に引き戻された瞬間、ジェイドは膝をつき、喉の奥から熱いものを吐き出した。

赤黒い血が床に散り、呼吸が乱れる。



「ジェイド！」

フィーネの声が響く。

しかしソルヴェンは動かず、その瞳だけが冷たく光っていた。



床に両手をついたまま、ジェイドは荒い息を整えた。

怖い。怒りが喉を灼く。悲しい――感情が互いを噛み合い、形を失っていく。



指輪が鼓動のように脈打ち、黒い燐光が指先から滲んだ。

その瞬間、塔の壁に刻まれた紋章がかすかに応じ、黒星の中心が“こちら”を見た気がした。

（やめろ……俺は、俺だ）

拒絶の念が溢れるたび、燐光は逆巻き、石床の上に細い亀裂を走らせる。



「おいジェイド、離れろ！」

キールが肩を掴みに来て、一瞬ためらった。目に走ったのは心配だけではない、恐れだ。

フィーネは震える声で詠唱し、透明な膜のような障壁を張る。「干渉を弱める……少しだけでも」

障壁に触れた黒い光がパチンと弾け、匂いのない火花が散った。



沈黙。

ソルヴェンは一歩だけ近づき、長い間、何も言わなかった。

その沈黙の重さに、耳鳴りさえ収まっていく。

やがて彼は低く呟く。「……触れるには早い。――いや、まだ“早くされている”のか」

言い切らない言い方が、不安だけを残す。



塔の奥から、気圧が変わるような圧が押し寄せた。

紋章の光がふっと消え、指輪の脈も静まる。

空気が切り替わった――誰もがそれを肌で感じた。



「帰還する」

ソルヴェンの声が落ちる。

「ここから先は“資格者”のみだ。今は違う」

ジェイドは立ち上がり、頷くしかなかった。

背を向けた階段で、フィーネが小さく囁く。「……混じってるのは、彼の中」

キールは言葉を飲み込み、ただ隣に立った。

三人の足音に、塔は何も答えなかった。



帰還路についた一行は、塔の階段をひたすら上り続けていた。

だが、足音以外のすべての音が吸い込まれたように消えている。

さっきまであった耳鳴りも、今は嘘のように静かだ。

それが余計に、不気味だった。



「……あの紋章、やっぱりただの装飾じゃねぇな」

キールが吐き捨てるように呟く。

フィーネは返事をせず、ただ前を見つめたままだった。

その耳は相変わらず微かに動き、遠くの何かを探るようだった。



やがて、第二階層の入口が見えた瞬間――

背後の奥底で、低くうねるような音が響いた。

地面が震え、壁に張り巡らされた魔法障壁が一瞬だけきしむ。

「……今の、聞こえたか？」

ジェイドが振り返るが、奥は闇に沈んで何も見えない。



ソルヴェンは一瞥だけ闇に送り、再び前を向いた。

「まだ崩れてはいない。だが……長くはもたん」

その声は淡々としていたが、わずかな焦りが混じっていた。



地上へ戻ると、夕陽が沈みかけていた。

訓練開始からの経過時間は、驚くほど短い。

それなのに、胸の奥には数日分の疲労が積もっていた。



「今回の探索記録は抹消される」

ソルヴェンは振り返り、全員を見渡した。

「理由は問うな。次に呼ばれる時まで、この件について口外するな」



ジェイドはその視線を正面から受け止めた。

あの紋章も、あの声も、そして血の匂いも――すべてが霧に包まれていく。

だが、その奥に潜むものは確かに呼んでいた。



（……必ず、もう一度行く）

心の奥底でそう誓い、ジェイドは塔を背に歩き出した。



後書き
精神干渉のシーンは、ジェイドの混乱や仲間との温度差を意識して描きました。

特に、ソルヴェンの沈黙や“資格者”発言は今後の大きな鍵になります。
