# 第5炎の審判話（初稿）

## 本文
Scene1　市場の朝（リライト／設定整合版）

中央市場。

朝の光が石畳に落ちている。

荷車の軋む音。
商人の呼び声。
焼いた肉の匂い。
香辛料の刺激的な香り。

市場はもう目を覚ましていた。

オレはその通りを歩く。

少し前を、姉ちゃんが歩いている。

赤橙色の髪。

朝の光を受けて、火みたいに揺れる。

「カーネル！」

果物屋のおじさんが声を上げた。

大きな箱を抱えて、困った顔をしている。

「悪い、これ運ぶの手伝ってくれ！」

姉ちゃんは迷わない。

「いいよ！」

箱の片側をひょいと持ち上げる。

「お前ほんと力あるな……」

おじさんが苦笑する。

姉ちゃんは肩をすくめた。

「パン屋だぞ」

「毎日粉袋運んでるんだ」

目的の場所まで運ぶと、おじさんが大きく息を吐く。

「助かった！」

姉ちゃんは軽く手を振る。

「ま、なんとかなる！」

その口癖に、周りの商人たちが笑う。

「相変わらずだな！」

「カーネル、今度うちも頼むぞ！」

「順番な！」

姉ちゃんは笑いながら歩き出す。

オレはその少し後ろを歩く。

市場の人たちは、みんな姉ちゃんを知っている。

声をかける。

笑う。

手を振る。

オレは思う。

……やっぱり姉ちゃんはすごい。

この市場じゃ、人気者だ。

少し歩くと、見慣れた木の看板が見えてくる。

粉屋クララ。

店の奥から声がした。

「カーネル」

クララだった。

粉袋を抱えたまま、こちらを見ている。

「また手伝ってたの？」

姉ちゃんは照れくさそうに笑う。

「まぁね」

クララは小さく息をつく。

「ほんとお人よし」

それから、オレを見る。

「ジェイド君も、おはよう」

「……おはよう」

クララは少し笑う。

「今日は粉？」

姉ちゃんが答える。

「うん、あとで取りに来る」

「了解」

クララは軽く手を振った。

オレたちはまた市場の通りへ戻る。

荷車が横を通る。

魚屋のドワーフが大きな魚を掲げて叫ぶ。

「港から届いたばかりだ！」

八百屋のエルフが籠を並べている。

「朝採れ野菜だよ！」

人間も。

エルフも。

ドワーフも。

入り混じって動く。

いつもの中央市場。

姉ちゃんが歩きながら言う。

「私さ」

「このあと教会」

オレは顔を上げた。

「また？」

姉ちゃんはうなずく。

「儀式の準備らしい」

「大事なんだってさ」

オレは市場の奥を見る。

石造りの建物。

小さな教会。

市場の喧騒から、少しだけ離れた場所に立っている。

姉ちゃんは軽く肩を回した。

「ちょっと手伝ってくる」

そして、いつもの笑い方で言う。

「ま、なんとかなる！」

手をひらひら振る。

そのまま教会の方へ走っていった。

白い布が、肩で揺れる。

オレはその背中をしばらく見ていた。

市場の声は、まだ賑やかだ。

荷車の音。

商人の呼び声。

笑い声。

いつもの朝。

それなのに。

ほんの一瞬だけ。

胸の奥が、ざわついた。

理由は分からない。

ただ。

教会の方を見たとき。

空気が、少しだけ歪んだ気がした。
