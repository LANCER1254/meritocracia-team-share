📐 運用ルール（制作・執筆共通）

・本作は「分かりやすさ」を目的としない
・読者に“理解させる”ことを目標にしない
・理解不能な部分が残ることを前提とする

・設定は説明されるために存在しない
・物語進行の必然としてのみ開示される

・キャラクターは思想の代弁者にならない
・思想は行動と選択によってのみ示される

🚫 禁止事項（厳守）

・主人公の万能化
・心音システムの能力化
・感情による問題解決

・制度側を「悪」と断定する描写
・ライナルトの矮小化・悪役化

・アイリスに思想を語らせる行為
・読者向けの答え合わせ台詞

✍ 執筆技法ガイド

・説明ではなく状況で示す
・会話は情報伝達より摩擦を優先
・正論同士をぶつける

・感情は爆発させない
・読者が不快になる余白を残す

・勝利の後に違和感を置く

・視点の一貫性

🧭 今後の執筆方針（最終）

・試験編は「イベント」ではなく「構造」として描く
・勝利と同時に必ず喪失を置く
・主人公自身が納得しない結末を選ぶ

・この物語に“答え”は存在しない
・進み続けること自体がテーマ

🔚 最終宣言

「正しさを否定する物語」ではない

「正しさだけでは足りない」
