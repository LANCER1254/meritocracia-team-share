# 【第04話】第四部
第四部｜後日談／余波／歪みの残滓
位置づけ（超重要）

第四部は
事件の解決編ではない。

「制度が正しかったあとに、
それでも残ってしまったもの」を描く部。

新しい事実は出さない

真相暴露もしない

誰も裁かれない

ただ、“歪みが残った”ことだけを確定させる

全体テーマ（第四部）

「終わったことにされた出来事は、
終わっていない場所に沈殿する」

プロット全体構成（案）
Scene 1｜余波（※導入）

役割：世界が“元に戻った”ことを示す

市場・街・日常は完全に復旧

人々はもう話題にしない

「事故だった」「もう大丈夫」

ジェイドの胸の“鳴り”は消えている

👉
事件が「完全に過去化」されたことを示す

Scene 2｜語られない会話

役割：大人の世界の“遮断”を描く

候補パターン（どれか一つでOK）：

大人同士の会話をジェイドが断片的に聞く

途中で会話が止められる

話題を変えられる

子どもが来た途端に沈黙

ポイント：

真相は語られない

でも「何かを隠している」ことだけは分かる

👉
“知らされない側”の位置づけを固定

Scene 3｜残された兆候

役割：心音システム／監視の余韻を残す

心音は鳴らない

でも「完全に消えた」とも言えない

視線・記録・偶然の一致

例：

誰かと目が合う

名前を呼ばれないまま確認される

数値・測定・“問題なし”

👉
第二部・第三部の制度的視線が、
　今も続いていることを示す

Scene 4｜父の不在（※重要）

役割：父親を“意図的に出さない”意味を持たせる

父親はまだ仕事

再会しない

連絡はあるかもしれないが、姿は出ない

👉
ここで父を出さないことで：

テロ事件後処理編との接続を守る

「まだ語られる段階に来ていない」ことを示す

Scene 5｜歪みの固定

役割：ジェイド側に残った“変化”を確定

世界は元に戻った

でもジェイドは“前と同じではない”

怒りでも復讐でもない

ただ「信じきれなくなった」感覚

例：

正しさを即座に信じなくなる

大人の言葉を一度置いて考える

胸の奥に「違和感」を保存する

👉
ここがジェイドの“起点”になる

Scene 6｜静かな終幕

役割：第一章（10歳編）への橋渡し

日常の一コマ

何も起きない

でも読者だけが分かる

「これは、
まだ始まりに過ぎない」
