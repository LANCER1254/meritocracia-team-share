# 【第04話】第四部
Scene構成（ep03後・確定）
Scene 1｜完全に戻った日常

役割：落差の提示

市場・街・学校は完全復旧

事件は「もう話題にされない過去」

周囲の大人も子どもも、普通に振る舞う

ジェイド視点で重要なのは：

何も変わっていない“ように見える”こと

それが 不自然に感じられる自分の存在

👉
ep03の重さと、世界の軽さの対比

Scene 2｜語られない“その後”

役割：沈黙の質の変化

ep03では「語られた」

第四部では「語られなくなった」

具体的には：

大人が事件の話題を避ける

「もう終わったことだから」という空気

話題を変える早さが不自然

ポイント：

隠しているのではない

終わったこととして扱っている

👉
制度的処理が、日常に浸透した証拠

Scene 3｜観測の継続

役割：第二・第三部の“制度”が今も生きていることの確認

心音は鳴らない

だが測定・視線・確認は続いている

「問題なし」という評価が繰り返される

重要なのは：

危険視されていない

でも 見られていないわけではない

👉
ジェイドは「自由」だが「放置」ではない

Scene 4｜父との距離

役割：ep03との対比による意味反転

ep03では父は「前に出ていた」

第四部では父は「日常に戻っている」

だが：

会話は噛み合っているようで、少しズレる

心配はされるが、踏み込まれない

「もう大丈夫だろ」という無意識の判断

👉
家族という安全圏が、完全ではないことの静かな提示

※ 不在ではない
※ ただ「完全な共有が終わった」

Scene 5｜歪みの定着

役割：ジェイド側の変化を確定

世界を疑うわけではない

反抗もしない

ただ、即座に信じなくなった

具体的には：

大人の言葉を一度置く

正しさをそのまま飲み込まない

胸の奥に違和感を“保存”する

👉
これが ジェイドという主人公の原点

Scene 6｜静かな橋渡し（第1章へ）

役割：プロローグの完全終幕

日常の一コマ

何も起きない

でも読者には分かる

この少年は
「世界を信じる側」から
「世界を見る側」へ移った

ここでプロローグは終わり。
第1章（10歳編）は この歪みを内包したまま始まる。
