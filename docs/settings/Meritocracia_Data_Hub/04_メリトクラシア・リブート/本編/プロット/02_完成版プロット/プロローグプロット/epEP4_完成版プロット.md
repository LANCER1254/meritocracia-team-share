# 【第EP4話】完成版プロット

📘 第二部 プロット化（Scene1〜6）
■ Scene1｜白い場所

爆発後、ジェイドは白い医療空間で目を覚ます

母も同じ空間で処置を受けている

機械的な医療確認（数値安定・異常なし）

状況説明は一切ない

「もう大丈夫」という言葉のみ繰り返される

胸の奥の違和感（未言語化）が残る

■ Scene2｜記録される

ジェイドは事情聴取を受ける

名前・年齢・位置・爆発音の確認

「異変」の項目はあるが、深掘りされない

胸の違和感は報告されない／聞かれない

記録は事務的に終了

「協力ありがとうございました」で締められる

胸の違和感は記録項目に存在しない

■ Scene3｜母という存在

母と休憩スペースで待機

母は泣いていないが動揺している

職員は身体的異常のみ確認

感情への介入はない

父への連絡のみ話題になる

胸の違和感は母にも共有されない

■ Scene4｜判断は既に下されている

別室へ移動

職員同士の報告会話が進行している

「心理的影響も想定範囲内」

同意書提示（保護措置終了・情報管理など）

署名で手続き終了

「対応は適切」「迅速」などの内部評価

事件は処理完了扱い

■ Scene5｜説明されない説明

担当者が状況説明

「危険は排除済み」

「原因は調査中」

「安心してください」

具体性はない

連絡先のみ渡される

胸の違和感については一切触れられない

■ Scene6｜処理の声

廊下で内部会話を聞く

「第二段階に移行」

「未成年は別枠」

「共鳴は弱い」

「正式記録は事故で統一」

「個人名は不要」

「統計処理」

「審問庁の管轄」

「事件性は低い／深追い不要」

胸の違和感が強まる

しかし説明はない

待機部屋で待たされる

物語は未解決のまま区切られる

爆発事件発生

国家主導の迅速処理

心理的影響は「想定範囲内」扱い

未成年は別枠処理

共鳴という概念が存在

公式記録は「事故」で統一

個別事例は統計処理

審問庁が最終管轄

主人公の内部異常は未記録
