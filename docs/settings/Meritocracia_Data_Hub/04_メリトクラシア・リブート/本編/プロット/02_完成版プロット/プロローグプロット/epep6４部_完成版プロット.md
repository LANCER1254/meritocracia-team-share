# 【第ep6４部話】完成版プロット

📘 第四部 プロット化
■ Scene1｜余波

市場は完全復旧済み

事件は「事故」として市民に共有済み

誰も話題を掘り下げない

社会は通常運転

ジェイドは違和感を抱くが明確化しない

心音は発生しない

「何もなかったこと」になっている構造を認識

■ Scene2｜語られない会話

家庭内も通常状態へ復帰

母は事件に触れない選択を取る

会話は日常話題のみ

「この前の…」と言いかけて止める

事件は家庭内でも処理済み扱い

ジェイドも踏み込まない

家族間で“語らない合意”が形成

■ Scene3｜残された兆候

学校も通常運転

事件の話題は出ない

非公開測定実施（器具・記録板）

数値結果：

中等

安定

異常なし

測定理由の説明なし

管理下にある事実のみ確定

心音発生なし

■ Scene4｜父の不在

父は帰宅せず

短文通知のみ

事件以降、父の直接言及なし

家庭は父不在を自然化

情報共有は未開始状態

不在は「異常」ではなく「未処理」

■ Scene5｜歪みの固定

世界を否定しない

事件への怒り・復讐感情なし

正義感も発生しない

しかし大人の言葉を即時信用できなくなる

信頼構造に微細な距離が生じる

違和感を保留する選択

世界を見る側へ立場移行

■ Scene6｜静かな終幕

日常完全復帰

物理的変化なし

内面的立ち位置のみ変化

世界と距離を取った観察者ポジション確立

決意なし・誓いなし

「事件後」ではなく「新しい日常の開始」と定義

物語は静的に一区切り
