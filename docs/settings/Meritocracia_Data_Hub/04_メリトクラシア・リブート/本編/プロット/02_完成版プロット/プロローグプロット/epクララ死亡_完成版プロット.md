# 【第クララ死亡話】完成版プロット

【第ep05話】クララ死亡
事実のみプロット化
■ 前提

クララはテロ事件で死亡

ジェイドは死体安置所で初めて死亡を認知

以後、制度による観測・記録が行われる

2年間墓参りを継続

Phase1｜事件後の日常

市場は修復済み

石畳・露店は整っている

住民は事件を「事故」として認識

粉屋の前を通過

入店しない

声を掛けない

Phase2｜家庭内

母は事件に触れない

「この前の……」と言いかけて止める

父は不在（仕事）

食事は通常通り

ジェイドは自室へ戻る

Phase3｜観測開始

校門前で見知らぬ大人に健康確認される

睡眠

胸の違和感

端末に記録

「問題ありません」と判断

継続観測状態

Phase4｜死亡確認導線

放課後呼び出し

「テロ被害者の確認が必要」と告知

機械じかけの馬で移動

市場を通過

粉屋は閉鎖

Phase5｜死体安置所

白い建物

長い廊下

泣き声と事務的指示が同時進行

名前が順番に呼ばれる

ジェイドも案内される

Phase6｜名簿確認

名簿提示

クララ・ヴェルナーの名前確認

端末に波形反応記録

「問題ありません」と処理

Phase7｜遺体確認

白布下の顔確認

粉の匂いを認知

波形変動記録

白布が戻される

「確認完了」と処理

Phase8｜崩壊

扉を叩く

名前を呼ぶ

泣く

制止される

Phase9｜事後説明

白い部屋

事故と説明

召喚獣による物理損壊

魔術的修復処置あり

不可逆損傷あり

「急性反応」「精神崩壊なし」と評価

経過観察対象

Phase10｜帰宅

母が迎えに来ている

会話ほぼなし

家へ戻る

ベッドで眠れない

名前を口内で反芻

Phase11｜二年後

定期的に墓参り

一人で訪問

花を供える

名前を声に出さない

日常生活継続

試験・評価・順位付け通常進行

観測継続示唆

「目を閉じない」姿勢の形成
