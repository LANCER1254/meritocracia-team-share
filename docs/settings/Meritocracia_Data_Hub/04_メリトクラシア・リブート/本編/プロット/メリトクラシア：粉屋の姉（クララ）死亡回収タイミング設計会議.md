メリトクラシア：粉屋の姉（クララ）死亡回収タイミング設計会議

■ 決定日時

2026-02-03

■ 作業者

Lancer

■ 対象範囲

プロローグ三部作～本編初期（士官学校入学直前）

1️⃣ 議題

粉屋の姉「クララ・ヴェルナー」の死亡回収タイミングと演出配置の最適化

2️⃣ 結論（確定）
■ 回収タイミング

アイリスとの主従手続き完了後
かつ
士官学校入学直前

このタイミングで、
クララ死亡に関する感情的回収（墓参り＋ユミナの涙）を行う。

3️⃣ 判断理由
① 物語構造上の整合性

過去：

クララ（初恋・日常側・守れなかった存在）

現在：

アイリス（守ると誓った存在）

未来：

士官学校（実力主義・選別の世界）

この3点を一本の線で接続できる。

② ジェイドの精神状態との適合

まだ子供であり、未熟

しかし「選択」は既に行っている段階

強くなりすぎていないため、
過去の喪失を受け取る余地がある

→ 回収が「遅すぎない」「早すぎない」。

③ ユミナの行動原理との整合

クララ死亡直後のユミナは

精神的に職務不能

審問庁・軍医・精神科などで隔離／静養状態

よって
テロ直後にジェイドと感情的に関わるのは不自然

2年後であれば、

表向きは立ち直っている

感情を封じ込めている

しかし未処理の罪悪感は残存

→ 墓参り目撃による感情決壊が自然に成立。

4️⃣ クララ死亡の扱い方針（再確認）
■ やらないこと（確定）

ジェイドの目の前で殺さない

直接的な殺害描写をしない

クララ自身に「死の自覚」を与えない

遺言・最期の言葉を描かない

■ やること（確定）

ジェイドは

死を「結果」として後から知る

葬儀には間に合わない

市場／粉屋の欠落によって
不在が日常に滲み出る構造を採用

5️⃣ 本編回収シーン概要（保存用）
■ 構図

ジェイド（10歳）

定期的にクララの墓参りをしている

誰にも言っていない

何を話しているかは描写しない

ユミナ

偶然、墓参りを目撃

最初は他人の少年として認識

墓標・仕草から身元を理解

■ 感情描写方針

ユミナは泣こうとしない

説明しない

ただ衝動的に抱きしめる

台詞（方向性・仮）：

「……ごめんなさい」
「少しだけ……こうさせて」

6️⃣ 構造的効果（確認）

クララ
→ 何も知らずに失われる

ジェイド
→ 知らないまま愛し、知らないまま失う

ユミナ
→ 知ってしまい、守れなかった

2年後、

唯一「失ったあとも続けていた人間」がジェイドだった

それを目撃したことで、
ユミナは初めて感情を解放できる

7️⃣ 総評（保存用）

本配置により、

初恋

喪失

罪悪感

許し

世代差

を一切の説明なしで同時回収可能。

物語初期段階における
最大級の感情的支柱シーンとして機能する。
