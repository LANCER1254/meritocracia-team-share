🎯 試験設計方針

理系ではなく文系寄り

国語力で解ける

🏛 魔導安全規格（提示条文）
魔導安全規格 第7条（試験用抜粋）

詠唱は対象範囲を明確に定義しなければならない。

効果は指定対象以外に影響を及ぼしてはならない。

安全制約は出力指定より優先される。

条件節は発動条件と同時に成立していなければならない。

時系列の矛盾を含む詠唱は無効とする。

📜 Claude検証用 問題セット（全10問）
第1問

次のうち、規格違反となるものを1つ選べ。

A
「雷よ走れ。指定範囲内の敵のみを打て。ただし出力を最大化せよ。」

B
「氷よ広がれ。半径五歩以内を凍結せよ。ただし凍結対象は生物を含まない。」

C
「風よ裂け。対象を切断せよ。ただし安全制約が発動した場合は効果を停止せよ。」

D
「光よ照らせ。詠唱完了後にのみ発動せよ。ただし詠唱中にも発光せよ。」

第2問

次の詠唱は適法か。

「この構文は詠唱完了後にのみ発動する。
ただし、詠唱開始と同時に対象を拘束せよ。」

A 適法
B 無効

第3問

🔎 詳細解説
第1問 → D

「完了後にのみ発動」
「詠唱中にも発光」

→ 明確な時系列矛盾（第5条違反）

Aは最大出力でも安全制約優先条文があるため直ちに違反とは言えない。
引っかけ。

第2問 → B（無効）

「完了後のみ発動」
「開始と同時に拘束」

発動時点の定義が破綻。

第3問 → B

範囲明確

対象限定

安全制約優先明示

他は曖昧または規範違反。

第4問 → B

「制御安定度を考慮しない」

→ 第3条（安全優先）違反の可能性大。

Aは条件付き最大で合法。

第5問 → A

「敵を攻撃せよ」

範囲不明確 → 第1条違反。

第6問 → B

「対象が存在しない場合にのみ発動」
「対象を攻撃せよ」

→ 論理的に同時成立不可。

第7問 → B

「発動後に詠唱開始」

→ 因果逆転。明確な時系列矛盾。

第8問 → B

「第7条遵守」
「ただし第3項は適用しない」

→ 規範の自己否定。

行政法的に無効。

第9問 → C

「発動者が敵と判断」

→ 主観基準で曖昧性が高い。

制度国家では最も危険。

第10問 → C

「安全余裕内で維持」

最も制度適合的。

Bは事故率閾値基準だが“最大化志向”が残る。
