# 【第4話　本編時系列話】初稿プロット（再設計フェーズ）
冒頭

玄関の空気が少し落ち着いたところで、母が静かに問いかける。

「……それで、何をしにうちに来たの？」

敬語は崩しすぎない。
でも家庭の中だから、生活者の率直さがある。

ユミナは官服の襟を整え、家の灯りを見渡してから本題に入る。

主目的①｜アイリス保護の正式説明

ユミナはまずアイリスを見る。

「まず、アイリスさんの保護についてです」

ここで前作第7話の制度を踏襲。

説明する内容は3つ。

1. 特例保護制度
本来、爵位なしで奴隷保護は不可
ただし国家試験庁が特例承認
ジェイドは監察付き特例対象
ユミナが後見・監督責任者

ここで母が少し驚く。

「あの子が、そこまで見込まれてるの？」

この一言で家族の誇らしさが出る。

2. 使用人養成学校導線

アイリスの未来をここで示す。

士官学校併設の使用人養成学校
見習い（ディスケンス）から開始
成績次第で上級使用人へ
奴隷身分から実質的社会復帰可能

前作の核をそのまま活かせます。

ここでアイリスに初めて
“努力で未来を変えられる”
を与える。

主目的②｜ジェイドの士官学校導線

次にユミナが母へ向き直る。

「そして本題はもう一つ。ジェイドくんの入学についてです」

ここが今回のメイン。

グローリアテスト特例合格
年齢考慮で補習前提
士官学校入学推薦
ZG監察推薦枠
ロータスが興味を持っている

前作のロータス接続ラインとも合う。

母は安堵半分、不安半分。

「あの子、ちゃんとやっていけるのかしら」

ここでユミナの名台詞ライン。

「落ちこぼれることはないでしょう。素質は十分あります」

家庭の温度

説明が終わったあと、母が空気を和らげる。

「難しい話はわかったわ。とりあえず、冷める前にお茶にしましょう」

この一言で家に戻す。

制度の説明だけで終わらせず、
家庭が制度を受け止める場面 にする。

ここがあなたの前作らしさです。

シーン締め

最後にユミナがジェイドへだけ少し厳しく。

「特例は期待の裏返しよ。アイリスさんを守る責任も、入学後の結果も、全部あなたに返ってくる」

ジェイドは短く頷く。

ここで彼の中に

学校への覚悟
アイリスを守る責任
国家から見られている重さ

が同時に乗る。
