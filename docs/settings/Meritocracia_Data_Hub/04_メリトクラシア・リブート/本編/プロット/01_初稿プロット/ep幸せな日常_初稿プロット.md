# 【第幸せな日常話】初稿プロット（再設計フェーズ）
📘メリトクラシア ZERO編
第2話プロット
■話数テーマ
幸せな日常

姉カーネルと市場の人々との交流を通して
ジェイドの生活環境を描く回。

物語の後半で起きる市場テロとの
対比となる日常描写が目的。

■登場キャラクター

ジェイド・レオンハルト
カーネル・レオンハルト
ガレス・レオンハルト（父）
ミレイユ・レオンハルト（母）
クララ・ヴェルナー（粉屋）

市場の商人たち（モブ）

■舞台

都市中央市場

特徴

・屋台
・粉屋
・肉屋
・パン屋
・貴族馬車が通る街道

後の

市場テロ事件

の舞台となる場所。

■シーン構成
Scene1
朝のレオンハルト家（コメディ）

ジェイドはまだ寝ている。

カーネルが部屋に突撃。

布団を剥がしながら

「起きろジェイドー！」

抱きついて起こす。

ジェイド抵抗。

カーネルはジェイドの初恋相手である
クララの名前を出してからかう。

ジェイド赤面。

家族の朝の雰囲気が描かれる。

Scene2
パン屋の手伝い

ジェイドは家業のパン屋を手伝う。

仕事内容

・配達
・粉運び

粉屋のクララの店へ
小麦粉を取りに行くことになる。

カーネルも一緒に市場へ向かう。

Scene3
市場の描写

中央市場は朝から賑わっている。

香辛料の匂い
商人の呼び声
屋台
荷車

カーネルは市場の人々に声をかけられる。

彼女は面倒見が良く
荷物運びなどをよく手伝うため

市場の人気者

として知られている。

Scene4
クララの粉屋

ジェイドは粉を取りに行く。

クララ登場。

クララは優しくジェイドに声をかける。

「今日はお使い？」

カーネルが茶化す。

「ジェイドこの人好きなんだよ」

ジェイド否定。

クララは笑ってジェイドの頭を撫でる。

ジェイドは照れる。

クララはジェイドにパンを渡す。

「頑張ってるからご褒美」

Scene5
貴族社会の伏線

市場の通りを

貴族の馬車が通る。

兵士が道を開け

平民たちは自然に道を譲る。

ジェイドは不思議そうに見る。

カーネルは

「貴族様だよ」

と軽く説明する。

Scene6
カーネルの教会バイト

市場の角で

カーネルはジェイドと別れる。

カーネル

「今日は教会の準備あるんだ」

仕事内容

・掃除
・儀式準備

ジェイド

「儀式？」

カーネル

「なんか大事らしいよ」

カーネルは手を振りながら

教会の方向へ去っていく。

Scene7
小さな違和感（能力伏線）

ジェイドは一瞬だけ

胸騒ぎのような感覚を覚える。

教会の方を見る。

しかし理由は分からない。

心音解析能力の伏線。

Scene8
夕方の帰り道（しっとり）

夕方。

市場の人通りが減る。

ジェイドとカーネルは帰り道。

カーネルはジェイドの頭を撫でる。

「ジェイドなら大丈夫」

抱きしめる。

ジェイドは照れる。

その時のジェイドは

この日常が
ずっと続くものだと思っていた。
