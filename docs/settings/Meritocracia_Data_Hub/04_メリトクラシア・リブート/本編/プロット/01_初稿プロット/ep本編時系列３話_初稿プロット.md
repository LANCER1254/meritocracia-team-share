# 【第本編時系列３話話】初稿プロット（再設計フェーズ）

Scene27 プロット

現在に戻った涙

部屋に残っていたのは、紙の擦れる音だった。

ユミナのペン先が、
最後の記録を一行だけ書き足す。

その小さな音で、
アイリスの意識がようやく現在へ戻る。

審問庁の応接室。

灰色の壁。
机の上のランプ。
向かいに座るユミナ。

隣にはジェイドがいる。

さっきまで見えていた家も、
荷馬車も、
冷たい値札も、もうない。

けれど呼吸だけはまだ浅い。

アイリスの頬を、涙が静かに伝う。

声は震えている。

でも、言葉は思ったより整っていた。

「……父も、母も」

小さく息を吸う。

「わたしを名前で呼びませんでした」

ジェイドがわずかに目を上げる。

ユミナのペンは止まらない。

「忌み子って……家の恥だって」

涙は落ちる。

けれど叫ばない。

崩れない。

むしろ、
感情を出し切った後みたいに声だけが冷えている。

「オーク帝国の人たちも……同じでした」

その一言で、
部屋の空気が少しだけ変わる。

アイリスは膝の上で手を握る。

「家族を、人じゃなくて……部位で見てた」

耳。
血。
目。
肌。

値段になる場所だけを見ていた。

思い出すだけで喉が震える。

それでも言葉は止まらない。

「……わたしだけ、生き残ったんじゃなくて」

一拍。

涙が机へ落ちる。

「売るために、残されたんです」

部屋が静かになる。

ユミナはそこで初めて、
記録の手を止める。

ジェイドは何も言えない。

ここで慰めを入れると軽くなる。

だから誰も、
すぐには言葉を返さない。

その沈黙の中で、
アイリスがもう一つだけ本音を落とす。

「……でも」

小さい。

壊れそうなくらい小さい。

「みんながいなくなった時……少しだけ、息がしやすかったんです」

空気が止まる。

善悪の言葉はどこにもない。

ただ、
悲しめなかった少女の本音だけが置かれる。

ジェイドはその横顔を見る。

助けを求めていただけの少女じゃない。

静かな地獄を生き抜いてきた人間なんだと、
初めて理解する。

ユミナは何も評価しない。

ただ、その一文を記録する。

“家族喪失に対する安堵反応あり”

それだけ。

制度は感情を裁かない。

ただ、残す。

涙のあとで触れる手

アイリスの最後の一言が落ちたあと、
部屋はしばらく静かだった。

「……少しだけ、息がしやすかったんです」

その言葉を言い切った瞬間だった。

ずっと保っていた呼吸が、
急に崩れる。

肩が震える。

膝の上で握っていた指がほどける。

そして次の瞬間、
抑えていたものが一気に溢れた。

「……っ、ぅ……」

声にならない。

涙だけが先に落ちる。

呼吸がうまくできない。

嗚咽が遅れて追いつく。

今まで淡々と話していた分、
反動が大きい。

家族の死。
売られるために残された自分。
悲しめなかった罪悪感。

全部が今になって胸を押し潰してくる。

ジェイドは言葉を探す。

でも、見つからない。

ここで正しい言葉なんてない。

先に動いたのはユミナだった。

椅子を静かに引く。

机を回り込み、
アイリスの横へ立つ。

何も言わない。

ただ、そのまま優しく抱き寄せる。

アイリスの身体が一瞬だけ強張る。

値札を付けられる時の手。
檻へ押し込まれる時の手。
命令を刻まれる時の手。

これまで触れてきたのは、
そういうものばかりだった。

でも違う。

ユミナの腕は強制しない。

押さえつけない。

ただ、逃げなくていい形で包む。

その温度に触れた瞬間、
アイリスの感情が完全に崩れる。

「……ぁ……あ……っ」

泣き声が漏れる。

肩を震わせ、
ユミナの制服に涙を落とす。

ユミナは髪を撫でる。

一定のリズム。
子供を落ち着かせるみたいに。

「もう、話さなくていい」

短い一言。

制度の声じゃない。

年上の女性の声だった。

アイリスはそのまましばらく泣き続ける。

ジェイドはその光景を見ている。

助けるという言葉より先に、
人はこうやって触れて支えることがある
と知る。

しばらくして呼吸が落ち着く。

ユミナはそっと身体を離す。

そのタイミングで、
机の上の書類をジェイドの方へ滑らせる。

ここで初めて紙に移る。

感情を一度受け止めたあとだから、
署名の重さが“冷たい制度”ではなく
守るための現実手続き
に変わります。
