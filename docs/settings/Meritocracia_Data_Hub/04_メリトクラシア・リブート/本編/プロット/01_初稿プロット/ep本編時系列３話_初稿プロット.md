# 【第本編時系列３話話】初稿プロット（再設計フェーズ）

Scene27 プロット

現在に戻った涙

部屋に残っていたのは、紙の擦れる音だった。

ユミナのペン先が、
最後の記録を一行だけ書き足す。

その小さな音で、
アイリスの意識がようやく現在へ戻る。

審問庁の応接室。

灰色の壁。
机の上のランプ。
向かいに座るユミナ。

隣にはジェイドがいる。

さっきまで見えていた家も、
荷馬車も、
冷たい値札も、もうない。

けれど呼吸だけはまだ浅い。

アイリスの頬を、涙が静かに伝う。

声は震えている。

でも、言葉は思ったより整っていた。

「……父も、母も」

小さく息を吸う。

「わたしを名前で呼びませんでした」

ジェイドがわずかに目を上げる。

ユミナのペンは止まらない。

「忌み子って……家の恥だって」

涙は落ちる。

けれど叫ばない。

崩れない。

むしろ、
感情を出し切った後みたいに声だけが冷えている。

「オーク帝国の人たちも……同じでした」

その一言で、
部屋の空気が少しだけ変わる。

アイリスは膝の上で手を握る。

「家族を、人じゃなくて……部位で見てた」

耳。
血。
目。
肌。

値段になる場所だけを見ていた。

思い出すだけで喉が震える。

それでも言葉は止まらない。

「……わたしだけ、生き残ったんじゃなくて」

一拍。

涙が机へ落ちる。

「売るために、残されたんです」

部屋が静かになる。

ユミナはそこで初めて、
記録の手を止める。

ジェイドは何も言えない。

ここで慰めを入れると軽くなる。

だから誰も、
すぐには言葉を返さない。

その沈黙の中で、
アイリスがもう一つだけ本音を落とす。

「……でも」

小さい。

壊れそうなくらい小さい。

「みんながいなくなった時……少しだけ、息がしやすかったんです」

空気が止まる。

善悪の言葉はどこにもない。

ただ、
悲しめなかった少女の本音だけが置かれる。

ジェイドはその横顔を見る。

助けを求めていただけの少女じゃない。

静かな地獄を生き抜いてきた人間なんだと、
初めて理解する。

ユミナは何も評価しない。

ただ、その一文を記録する。

“家族喪失に対する安堵反応あり”

それだけ。

制度は感情を裁かない。

ただ、残す。

cene28 改訂プロット

涙のあとで触れる手

アイリスの最後の一言が落ちたあと、
部屋はしばらく静かだった。

「……少しだけ、息がしやすかったんです」

その言葉を言い切った瞬間だった。

ずっと保っていた呼吸が、
急に崩れる。

肩が震える。

膝の上で握っていた指がほどける。

そして次の瞬間、
抑えていたものが一気に溢れた。

「……っ、ぅ……」

声にならない。

涙だけが先に落ちる。

呼吸がうまくできない。

嗚咽が遅れて追いつく。

今まで淡々と話していた分、
反動が大きい。

家族の死。
売られるために残された自分。
悲しめなかった罪悪感。

全部が今になって胸を押し潰してくる。

ジェイドは言葉を探す。

でも、見つからない。

ここで正しい言葉なんてない。

先に動いたのはユミナだった。

椅子を静かに引く。

机を回り込み、
アイリスの横へ立つ。

何も言わない。

ただ、そのまま優しく抱き寄せる。

アイリスの身体が一瞬だけ強張る。

値札を付けられる時の手。
檻へ押し込まれる時の手。
命令を刻まれる時の手。

これまで触れてきたのは、
そういうものばかりだった。

でも違う。

ユミナの腕は強制しない。

押さえつけない。

ただ、逃げなくていい形で包む。

その温度に触れた瞬間、
アイリスの感情が完全に崩れる。

「……ぁ……あ……っ」

泣き声が漏れる。

肩を震わせ、
ユミナの制服に涙を落とす。

ユミナは髪を撫でる。

一定のリズム。
子供を落ち着かせるみたいに。

「もう、話さなくていい」

短い一言。

制度の声じゃない。

年上の女性の声だった。

アイリスはそのまましばらく泣き続ける。

ジェイドはその光景を見ている。

助けるという言葉より先に、
人はこうやって触れて支えることがある
と知る。

しばらくして呼吸が落ち着く。

ユミナはそっと身体を離す。

そのタイミングで、
机の上の書類をジェイドの方へ滑らせる。

ここで初めて紙に移る。

感情を一度受け止めたあとだから、
署名の重さが“冷たい制度”ではなく
守るための現実手続き
に変わります。


Scene29 プロット

湯気の向こうの名前

泣き声が少し落ち着いた頃。

応接室の扉が、
控えめに二度だけ叩かれる。

「失礼します」

入ってきたのは、白に近い淡灰色の制服を着た女性保護係だった。

年齢はユミナより少し上。

落ち着いた手つきで、
銀の小さなトレイを机に置く。

ティーポット。
白磁のカップ。
湯気。

冷えていた部屋の空気が、
その一角だけ少し柔らかくなる。

アイリスはまだユミナのそばで、
浅い呼吸を繰り返している。

女性保護係は何も詮索しない。

ただ静かにカップへ茶を注ぐ。

淡い金色。

花の甘い香りに、
少しだけ柑橘が混じる。

アイリスの睫毛がわずかに揺れる。

「カモミールに、ベルガモットを少し」

女性保護係が穏やかに言う。

「緊張をほどくための配合です」

ジェイドが小さく視線を向ける。

彼女は続ける。

「カモミールに含まれるアピゲニンは、張り詰めた神経を少し緩めるんです」

さらりとした説明。

医療でも尋問後の保護でも使われる、
この国では半ば慣例の処置。

制度は冷たい。

でも壊れたまま放置はしない。

それがメリトクラシアのやり方だった。

女性保護係はカップを、
アイリスの前へそっと置く。

「ベルガモットを入れたのは、あなたの名前に少し似せて」

アイリスの目が、
初めてカップへ向く。

アールグレイ。

自分の名前に含まれた言葉。

これまでは、
値札や商品台帳でしか見なかった文字。

それが今は、
湯気の向こうで柔らかい香りになっている。

その違いに、
アイリスは少しだけ言葉を失う。

恐る恐るカップを持つ。

まだ指先は震えている。

けれど温かい。

唇を寄せる。

花の甘さ。
柑橘の香り。
熱が喉を通って落ちていく。

浅かった呼吸が、
少しだけ深くなる。

女性保護係はそれを確認し、
小さく頷く。

ユミナは何も言わない。

ただ、その変化を見ている。

ジェイドもまた、
言葉を挟まない。

助ける言葉ではなく、
こうして温度で支える方法がある。

その空気を覚える。

部屋の沈黙はもう重くない。

湯気が、
灰色の応接室を少しだけ人の部屋に変えていた。

Scene30 プロット

覚えていた妹

紅茶の湯気が、まだ細く揺れている。

少しだけ空気が和らいだところで、
ユミナがふとジェイドへ視線を向ける。

今度は業務の顔じゃない。

少しだけ柔らかい声。

「ジェイド」

呼ばれて、ジェイドが顔を上げる。

ユミナは小さく首を傾ける。

「……私のこと、覚えてる？」

唐突な問いに、
ジェイドは一瞬だけ戸惑う。

「え？」

どこかで見た顔。

今日の掲示板。
試験会場。
黒衣。

違う。

もっと前。

市場。
黒布。
棺。
泣き崩れる人影。

そこで記憶が繋がる。

「あ……」

短く息を呑む。

「もしかして……あの、俺の姉さんの」

ユミナが小さく頷く。

「そう」

視線が少しだけ遠くなる。

「あなたのお姉さん、カーネルの葬式のときに居ました」

応接室の空気が静かに市場へ繋がる。

ジェイドの中でも、
あの日の石畳の色が戻る。

棺の前。
崩れ落ちるクララ。
その肩を支えていた少女。

今日アイリスを抱きしめた腕と、
同じ優しさだった。

ユミナはそこで少しだけ苦笑する。

「姉のクララ、元気？」

ほんの少しだけ、
仕事の顔が消える。

「最近忙しくて、なかなか見に行けてないの」

ZGの監察主任としてではなく、
ただの妹の言葉。

ジェイドも少しだけ肩の力を抜く。

「元気だと思う」

正確な断言はしない。

でも市場の店先で見た、
あの働きぶりなら想像できる。

「相変わらず忙しそうだった」

ユミナがそれを聞いて、
小さく安堵する。

その会話を、
アイリスは静かに見ている。

カップを持つ手が少しだけ止まる。

姉。
妹。
元気？

そんな何気ない会話。

自分の世界には、
もう存在しない言葉だった。

アイリスは恐る恐る口を開く。

「……ご主人様、お姉様がいたんですね……」

言いかけて、止まる。

喉が詰まる。

その先に続くはずだった言葉は、
うまく形にならない。

でも、その。

家族という響きに触れた瞬間、
胸の奥でさっきの過去がまた少しだけ疼く。

羨ましいのか。
怖いのか。
悲しいのか。

まだ自分でも分からない。

だから言葉が続かない。

ジェイドはその言い淀みを見て、
すぐには返さない。

ここで無理に埋めない。

ユミナもまた、
その沈黙の意味を理解している。

家族の世間話が、
この部屋で一番残酷で、一番優しい話題になっていた。

審問庁聴取室の公的空間感を補強するため、背景に書記官・女性保護係・扉前ZG兵を追加。会話主軸はアイリス／ジェイド／ユミナの3人で維持。
