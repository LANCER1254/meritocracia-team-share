第一部「生き残り」
完成版プロット（再構築確定）
Scene 1｜兆し（Before）

目的（再定義）
日常が「平和だった」と言い切れない温度を残す。

構成

人の多さ、音の重なり、足音の反響

母は少し早足／人混みを嫌う

父は視界にいない（理由は語らない）

ジェイドは「特別な違和感」は覚えていない

固定感覚

音が多い

うるさいのではなく、整理されていない

👉
後から振り返ったとき、
「何もなかった」と言えない空気だけを残す。

Scene 2｜すれ違い（ユリス）

目的（再定義）
“加害者”を、人間として記憶させる。

構成

顔色が悪く、呼吸が荒い男

手の震え／何かを握っているような仕草

一瞬、目が合う

会話なし

ジェイドの印象（重要）

怖い

でも「悪そう」ではない

自分と同じように、追い詰められている感じ

👉
ここでの違和感は
**「人間としての不安定さ」**に限定する。
（制度・超常・心音とは切り離す）

Scene 3｜爆心（破壊）

目的（固定）
理解不能な暴力を「現象」として体験させる。

構成

光が先、音が後

世界が反転する感覚

身体が宙に投げ出される

痛みは遅れて来る

禁止

事件名

意図

誰かの叫びによる説明

👉
“何が起きたか分からない”状態を最優先。

Scene 4｜崩壊（分断）

目的（修正）
「考える」前に、生き残る選択を迫られる。

構成

視界不良／埃／煙

母が倒れている（軽傷か重傷かは不明）

動かない人がいる（詳細描写なし）

ジェイドの内面（重要修正）

「考えろ」ではない

何かしなきゃいけない気がした
でも、何をすればいいのか分からなかった

👉
無力さを固定する。
ここではまだ、主体的な判断をさせない。

Scene 5｜介入（FO）

目的（固定）
秩序が、感情抜きで介入する瞬間。

構成

揃った足音

無線

短い指示

名前の出ない指揮官

固定セリフ

「……生きたいか、少年」

重要

慰めない

説明しない

正義を語らない

👉
ここで提示されるのは
価値観ではなく“条件”。

Scene 6｜即時排除（〈刃〉）

目的（固定）
暴力＝迷わないこと、という誤解を刻む。

構成

ユリスが錯乱状態で現れる

叫び／逃走

剣士が一歩出る

一太刀

終了

描写方針

技の美化なし

血の強調なし

速すぎて理解が追いつかない

ジェイドの認識（固定）

「あの人が、あの人を殺した」

実感は伴わない。

👉
この“実感の欠如”が
第二部・第三部で反転する。

Scene 7｜視線（ノクサ）

目的（再定義）
制度でも暴力でもない“未分類の違和感”。

構成

爆心地から少し離れた場所

立ち尽くす、もしくは誰かを助けようとする女

目が合う

感覚（心音と明確に分離）

整理されないノイズ

意味にならない重なり

「最適」でも「安全」でもない

ジェイドの印象

あの人も、何かを聞いていた気がする

でも、何も分からない

👉
ここでは
共鳴＝希望 にしない。

Scene 8｜切断（After）

目的（固定）
生き残ったことだけを、事実として残す。

構成

担架

医療班

FOは去る

描写の核

説明がない

謝罪がない

理由が与えられない

終了状態（固定）

ジェイドは生きている

母も生きている

世界は壊れたまま

強さとは
→ 迷わないこと
→ 殺すこと
だと、誤解したまま
