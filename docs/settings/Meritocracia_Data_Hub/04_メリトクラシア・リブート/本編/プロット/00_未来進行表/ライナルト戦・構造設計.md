# 未来進行表｜ライナルト戦・構造設計

📘 議事録｜ライナルト戦・構造設計（仮確定）
■ 作成目的

・ライナルト戦の形式確定
・チーム構成／観戦構造の整理
・キャラ配置と役割の明確化

■ 戦闘形式（確定）
● 基本構造

👉 3 vs 3 チーム戦（実質1vs1収束型）

● コンセプト

・表面：チーム戦
・本質：個の対決

👉 最終的に
ジェイド vs ライナルトに収束させる

■ チーム構成（ジェイド側）
● メンバー

① ジェイド
→ 主軸／理想

② キール
→ 現実／生存

③ フィーネ
→ 判断／戦術

● 構造

👉 思想三角形

理想 vs 現実 vs 判断
■ チーム構成（ライナルト側）
● 参戦メンバー

① ライナルト
→ ボス／支配構造

② 側近A
→ 魔術特化（理論型）

③ 側近B
→ 近接特化（暴力型）

● 非参戦（観戦）

④ 側近C
→ 分析／言語化担当／伏線枠

■ 戦闘構造（流れ）
● 序盤

・側近2名が前線
・ジェイド側分断

● 中盤

・キール単独戦（生存戦闘）
・フィーネが状況分析

● 転機

・ジェイド覚醒

● 終盤

👉 戦闘が収束

👉 ジェイド vs ライナルト

■ キャラ役割（戦闘内）
● ジェイド

・中心
・覚醒トリガー

● キール

・近接戦担当
・現実的戦い方提示

● フィーネ

・戦術／分析
・冷静な判断軸

● 側近A

・魔術戦
・フィーネと対比

● 側近B

・肉弾戦
・キールと対比

● 側近C（観戦）

・状況言語化
・異常検知
・後の伏線キャラ

■ 観戦構造（確定）
● 半公開試験形式

👉 「選ばれた者のみ観戦可能」

● 観戦者

・審問庁（ヴィオラ含む）
・近衛／教官
・三兵団代表（各1名程度）

● 不採用

・一般観客
・ライナルトの両親

● 役割

・評価
・監視
・記録

■ 使用人陣営（確定）
● 立場

👉 戦闘不参加

● 配置

・観戦 or 裏方

● 役割

・感情導線
