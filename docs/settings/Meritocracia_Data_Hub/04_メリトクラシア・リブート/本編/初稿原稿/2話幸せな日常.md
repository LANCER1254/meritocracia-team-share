# 第2話幸せな日常話（初稿）

## 本文
Scene1　朝のレオンハルト家（リライト版）

　あの不思議な少年と出会ってから、二年が過ぎていた。

　オレは八歳になった。

　この頃のオレは、まだ知らなかった。

　この家も。
　この市場も。
　この騒がしい朝も。

　――あと三ヶ月で、全部終わるなんて。

　そんなこと、考えたこともなかった。

　だからこの朝も、いつも通りだった。

　布団の中はまだ暖かい。

　外から聞こえるのは、パン屋の準備をする音。
　台所では母さんがパン生地をこねているらしく、香ばしい匂いが漂ってくる。

　……でも。

　眠い。

　もう少しだけ寝ていよう。

　そう思って布団に顔をうずめた――その瞬間だった。

　ばさっ！

　突然、布団が勢いよく剥がされた。

「起きろジェイドー！」

「うわっ！」

　冷たい朝の空気が一気に流れ込み、オレは思わず飛び起きた。

　目の前で揺れたのは、燃え盛る炎をそのまま髪にしたような鮮烈な橙。

　松明の火を思わせる赤橙色の髪。

　カーネル・レオンハルト。

　市場一元気なパン屋の娘。

　そして――オレの姉だ。

　カーネルは腰に手を当て、にやりと笑った。

「まだ寝てたのかよ」

「まだ朝だろ……」

「朝だから起きるんだろ」

　正論すぎる。

　オレが顔をしかめていると、カーネルはさらにニヤニヤしながら言った。

「クララさん来るかもしれないぞ？」

「なっ――！」

　顔が一気に熱くなる。

「やめろ！！」

　思わず叫ぶと、カーネルは腹を抱えて笑い出した。

「ははは！やっぱ反応するじゃん！」

「してない！」

「してるしてる！」

　言い合っていると、台所から母の声が飛んできた。

「朝から騒がないの」

　ミレイユ・レオンハルト。

　このパン屋の店主であり、オレたちの母さんだ。

「ジェイド、起きたなら顔洗ってきなさい」

「はーい……」

　まだ眠い目をこすりながら立ち上がる。

　部屋の隅では父さんが仕事の準備をしていた。

　ガレス・レオンハルト。

　朝が早い仕事で、もう出かけるところらしい。

「今日は市場、混むぞ」

　父さんは短く言った。

「荷車が三台入るらしい」

「まじ？」

　カーネルの目が輝く。

「手伝い増えるじゃん！」

　父さんが苦笑する。

「お前が張り切るとだいたい大変になる」

　カーネルは肩をすくめて言った。

「ま、なんとかなる！」

　それが姉の口癖だった。

　オレは顔を洗いながら、ぼんやり思う。

　パン屋の朝は、いつも騒がしい。

　でも。

　それが当たり前だった。

　この家も。
　この市場も。
　この笑い声も。

　――ずっと続くと思っていた。
Scene2　パン屋の手伝い

　レオンハルト家の一階は、そのままパン屋になっている。

　朝の店内には、焼きたてのパンの匂いが満ちていた。

　木の棚には丸パンや黒パンが並び、まだ湯気を立てている。外では市場へ向かう人たちの足音が聞こえ、扉が開くたびに冷たい朝の空気が入り込んできた。

　オレは大きな籠を抱えて、店の奥と入口を何度も往復していた。

　パン配達。

　小麦粉運び。

　店番の補助。

　八歳でもできる仕事は、意外と多い。

「ジェイド、その籠こっち」

「はい」

　母さん――ミレイユは手際よくパンを並べながら言った。

　店の奥では、次の生地がもう焼き上がろうとしている。

　忙しい朝だ。

　カウンターに籠を置いたとき、母さんがふと棚を見て言った。

「……粉が少ないわね」

　オレも棚を見た。

　小麦粉の袋が、いつもより減っている。

「昨日の分で結構使ったからね」

　母さんは軽く息をつく。
「クララさんところから、もう少し仕入れないと」
クララ。

　市場の粉屋の娘。（訂正必要）
名前を聞くだけで、なんだか胸のあたりがむずむずする。

「取りに行ってくる」

　オレが言うと、母さんはうなずいた。

「お願い」

　すると、すぐ横から声が飛んだ。

「私も行く」

　振り向くと、カーネルがパンを頬張りながら立っていた。

「お前食ってるだけだろ」

「エネルギー補給だよ」

　カーネルは平然と言う。

　赤橙色の髪が、朝の光に照らされて揺れていた。

「荷物持つの手伝ってやるよ」

「別に一人で持てる」

「どうせ途中で休むだろ？」

「休まない！」

　オレが言い返すと、カーネルはにやりと笑った。

「ま、なんとかなる！」

　それが姉の口癖だった。

　母さんは呆れたように笑う。

「ついでに市場の様子も見てきて」

「はーい」

　カーネルが軽く手を振る。

　店の扉を開けると、外の空気が一気に流れ込んできた。

　市場は、もう動き始めている。

　荷車の音。

　商人の呼び声。

　遠くで肉を叩く音。

　いつもの朝の市場だ。

　オレとカーネルは顔を見合わせた。

「行くぞ」

「うん」

　そうして二人で、中央市場へ向かった。
