# 第修正版第一話話（初稿）

## 本文
Scene1　悪夢

雨が降っていた。

冷たい雨だ。

空は暗く、雲は低く垂れ込めている。

足元の泥が、靴にまとわりつく。

その向こうに――

倒れている影があった。

ゆっくりと近づく。

呼吸は浅い。

血が、雨に流されていく。

そして。

背中から伸びるものが見えた。

黒い翼。

ぼろぼろに裂けている。

地面に広がった羽が、雨に濡れて重く沈んでいた。

その存在は、わずかに顔を上げる。

目が合った。

かすれた声。

「……来るな」

弱々しい声だった。

だが、はっきりと拒絶の色がある。

それでも足は止まらない。

「まだ助かる」

自然と口から言葉が出た。

理由は分からない。

ただ。

放っておけないと思った。

手を伸ばす。

黒い翼の存在が、わずかに眉をひそめた。

「……やめろ」

声は低い。

どこか、焦りが混じっていた。

「助ければ――」

言葉が途切れる。

その瞬間だった。

空が、光った。

雷ではない。

もっと鋭い光。

空そのものが裂けたような輝き。

風が止まる。

雨粒が、空中で静止したかのように感じた。

黒い翼の存在が目を閉じる。

そして、小さく呟く。

「……遅い」

光が落ちてきた。

空から。

一直線に。

地面へ。

その中心に、影が立つ。

白い翼。

静かな気配。

足音はない。

ただ、そこに降り立っていた。

視線が向けられる。

冷たい視線。

そして声。

「なぜ庇った」

静かな声だった。

だが、怒りを抑え込んでいる。

「……」

言葉を探す。

うまく出てこない。

ただ、胸の奥から出たのは。

「放っておけない」

それだけだった。

沈黙。

白い翼が、黒い翼を見る。

「それは敵だ」

短い言葉。

断定。

それでも首を振った。

「……違う」

理由は説明できない。

でも、違うと思った。

空気が凍る。

白い翼の存在が、ゆっくりと剣を抜いた。

光を帯びた刃。

冷たい光。

次の瞬間。

剣が振り下ろされた。

胸に衝撃が走る。

息が止まる。

視界が揺れる。

力が抜ける。

膝が崩れる。

地面が近づく。

雨が顔に当たる。

冷たい。

遠くなる意識の中で、最後に見えたのは。

黒い翼だった。

その存在が、こちらを見ている。

そして。

小さく呟いた。

「……だから言った」

視界が暗くなる。

雨の音だけが残る。

そして――

闇。
――っ！

息が詰まったみたいに、オレは体を起こした。

胸が苦しい。

心臓が、やけに速い。

ドクン。

ドクン。

ドクン。

しばらく呼吸が整わない。

……なんだ、今の。

周りを見る。

見慣れた天井。

古い木の梁。

少し曲がった板。

窓から朝の光が入っている。

雨は降っていない。

静かな朝だ。

オレは手を見る。

少し震えていた。

「……夢？」

小さくつぶやく。

頭の奥に、さっきの光景が残っている。

黒い翼。

光。

剣。

誰かの声。

でも――

思い出そうとすると、ぼやける。

変な夢だった。

知らない場所。

知らない誰か。

それなのに、妙に胸がざわつく。

オレは額の汗をぬぐった。

シャツが背中に張りついて気持ち悪い。

「……はぁ」

ため息をつく。

夢だ。

ただの夢。

そう思うことにした。

窓の外を見る。

朝の街。

パン屋の煙突から、白い煙が上がっていた。

いつもの景色。

それを見て、少し落ち着く。

そのとき。

階下から声が聞こえた。

「ジェイドー！」

……姉ちゃんだ。

「いつまで寝てるのー！」

ドタドタと階段を上がってくる足音。

オレは小さく息を吐いた。

どうやら今日は、普通の朝らしい。
Scene3　姉

ドタドタドタ。

遠慮のない足音が、階段を上がってくる。

……来た。

次の瞬間。

勢いよく扉が開いた。

「ジェイド！」

姉ちゃんだった。

カーネル・レオンハルト。

寝癖のついた赤い髪を揺らしながら、腰に手を当てて立っている。

「まだ寝てたの？」

呆れた顔。

オレはベッドの端に座ったまま、肩をすくめた。

「起きてる」

「それ、起きてるって言わない」

姉ちゃんは部屋の中をずかずか入ってくる。

窓のカーテンを勢いよく開けた。

朝の光が、一気に部屋に流れ込む。

まぶしい。

「うわ……」

思わず顔をしかめる。

姉ちゃんは振り返って笑った。

「ほら、朝だよ」

元気すぎる。

オレは頭をかいた。

「……変な夢見た」

ぽつりと言う。

姉ちゃんは首を傾げた。

「また？」

「うん」

「どんな夢？」

うまく説明できない。

黒い翼。

光。

剣。

誰かの声。

でも、思い出そうとすると、やっぱりぼやける。

オレは首を振った。

「覚えてない」

姉ちゃんは少し笑った。

「じゃあ気にしなくていいじゃん」

軽い。

いつも通りの反応だ。

「夢なんてそんなもんでしょ」

そう言って、オレの頭をぽんと叩く。

「ほら、さっさと着替えて」

「朝ごはん出来てるよ」

そして、くるりと背を向ける。

「あと五分で来なかったら――」

ちらっと振り返る。

にやっと笑った。

「全部食べるからね」

「……ひどい」

「早く来ればいいの」

姉ちゃんはそう言って、またドタドタと部屋を出ていった。

階段を下りる音が遠ざかる。

オレは少しだけ天井を見上げる。

さっきの夢。

黒い翼。

あの光。

胸の奥が、まだ少しだけざわついている。

「……変な夢」

小さくつぶやく。

でも。

もう思い出せなかった。

普通の日になりそうだった。
パンをかじっていると、奥の台所から声がした。

「カーネル、もう食べてるの？」

落ち着いた声。

母さんだった。

ミレイユ・レオンハルト。

エプロン姿で台所から出てくる。

手にはスープの鍋。

「ジェイドも起きたのね」

オレを見ると、柔らかく笑った。

「おはよう」

「……おはよう」

母さんはテーブルに鍋を置く。

ふわっと湯気が上がった。

「今日は早いのね」

姉ちゃんが笑う。

「ジェイドが変な夢見たって」

母さんが少し首を傾げる。

「夢？」

オレはパンをかじりながら答える。

「覚えてないけど」

母さんは少し考えてから言った。

「夢ってね」

「心が疲れてると見るのよ」

優しい声だった。

怒るでもなく、笑うでもない。

ただ、穏やかな声。

「無理してない？」

オレは首を振る。

「してない」

姉ちゃんが笑う。

「こいつ、朝からぼーっとしてるけどね」

「うるさい」

母さんは小さく笑った。

それだけだった。

いつもの朝。

パンの匂い。

温かいスープ。

姉ちゃんの笑い声。

母さんの穏やかな声。

さっきまで見ていた夢が、嘘みたいだった。

オレはスープを飲む。

温かい。

胸の奥のざわつきが、少しだけ消えていく。

そのとき母さんが言った。

「今日はいい天気になりそうね」

窓の外を見る。

朝の光が、街を照らしていた。

本当に――

ただの、いつもの朝だった。
