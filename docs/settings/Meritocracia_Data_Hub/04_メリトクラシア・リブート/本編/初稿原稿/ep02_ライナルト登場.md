## 📖 本文

Scene2-1｜掲示板の前

校庭の端に、人だかりができていた。

建物の壁に、白い紙が何枚も貼られている。
番号と名前が、縦に並んでいる。

ざわつきはあるが、大声はない。
確認して、離れる。
それだけの動きが、繰り返されている。

少し離れた場所に、別の掲示があった。
そちらの前にも、数人が集まっている。

見覚えのある背中があった。

銀に近い色の髪。
背筋がまっすぐで、動かない。

その横に、二人。

「あれ、レオンハルトじゃね？」

小さな声が飛んできた。

ジェイドは足を止める。

Scene2-2｜軽い絡み

「今日だろ？　本番」

子分の一人が、口元をゆがめる。

「初受験ってやつ」

ライナルトは、紙から目を離さない。
一拍遅れて、こちらを見る。

視線だけが動く。

「……そうか」

それだけ言って、また掲示に戻る。

ジェイドは肩をすくめる。

「そっちこそ、発表か？」

子分が笑う。

「当然だろ。今回で終わりだ」

ライナルトが、静かに口を開く。

「当然だ」

声は低い。
強くも弱くもない。

ジェイドはうなずく。

「なら、見る必要もないな」

一瞬だけ、子分が言葉に詰まる。

ライナルトの視線が、わずかに細くなる。

怒ってはいない。

「確認はする」

短く言う。

「それだけだ」

ジェイドはそれ以上何も言わない。

「そうか」

それだけ返す。

Scene2-3｜収束

子分の一人が、何か言いかける。

「おい——」

「やめろ」

ライナルトが止める。

視線はジェイドから外さない。

「受かるといいな」

嘲りではない。

評価でもない。

ただの事実確認みたいな声だった。

ジェイドは答える。

「そっちも」

沈黙が落ちる。

風が、紙を揺らす。

それぞれが、別の掲示へ向き直った。
