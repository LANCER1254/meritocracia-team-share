# 第3 ジェイドの淡い恋心話（初稿）

## 本文
Scene1
朝のレオンハルト家

朝。

まだ空気が少し冷たい。

店の中には、焼きたてのパンの匂いが広がっていた。

作業台の横に置いてある粉袋を持ち上げる。

軽い。

嫌な予感がして、中を覗き込んだ。

……やっぱり、ほとんど残ってない。

「……ちょっと少ない」

台所の奥から母さんの声がする。

「粉、まだ残ってたかしら？」

袋を軽く揺らす。

底に白い粉が少しだけ残っている。

「……これだけ」

母さんが眉をひそめた。

「昨日買ったばかりでしょ？」

「昨日忙しかっただろ」

母さんは少し考える。

「ああ……そうね」

昨日は昼からずっと客が来ていた。

袋をもう一度持ち上げる。

「すぐ無くなるよ」

そのやり取りを横で見ていたカーネルが、ニヤニヤしている。

嫌な予感がした。

「ジェイドさぁ」

やっぱりだ。

カーネルはわざとらしい声で言う。

「粉屋のお姉さんに会いに行くんでしょ？」

顔が一気に熱くなる。

「違う！」

カーネルは大笑いした。

「顔真っ赤ー！」

思わず視線を逸らす。

すると母さんが、くすっと笑った。

「クララさん、いい子よね」

「母さん！」

カーネルは腹を抱えて笑っている。

「ほら認めた！」

「認めてない！」

母さんは呆れたようにため息をついた。

「はいはい、二人とも」

そして棚の横にあった袋を手に取る。

「粉、買ってきて」

ぽすん、と腕に押しつけられた。

カーネルが横でニヤニヤしている。

「デート頑張ってー」

「違うって！」

パン屋の朝は、今日も騒がしかった。

Scene2
市場へ向かう

袋を肩にかけて家を出る。

朝の空気はまだ少し冷たい。

後ろから足音がついてきた。

振り返ると、やっぱり姉ちゃんだった。

「なんで来るんだよ」

「暇だから」

「絶対暇じゃないだろ」

姉ちゃんはにやっと笑う。

「デートの付き添い」

「違う！」

姉ちゃんは楽しそうに笑っている。

そのまま二人で中央市場へ向かった。

通りを曲がると、空気が一気に変わる。

人の声。

荷車の軋む音。

屋台から立ち上る煙。

そして香辛料の匂い。

市場は、いつも通り賑やかだった。

「安いよ安いよ！朝一の野菜だ！」

「焼きたてパン！」

「今日の川魚だ！」

商人の呼び声があちこちから飛ぶ。

果物の山。

肉屋の包丁の音。

荷車を押す人たち。

人の流れの中を歩きながら、俺は袋を肩に掛け直した。

その横で、姉ちゃんが手を振る。

「おはよー！」

すぐに声が返ってきた。

「おうカーネル！」

肉屋のおじさんだ。

別の店からも声が飛ぶ。

「今日も元気だな！」

「また手伝えよ！」

姉ちゃんは笑って手を振る。

「あとでね！」

市場を歩くたび、誰かが姉ちゃんに声をかける。

俺は少し呆れながら言った。

「顔広すぎだろ」

姉ちゃんは肩をすくめる。

「普通だよ」

普通じゃない。

姉ちゃんは、この市場ではちょっとした人気者だった。
