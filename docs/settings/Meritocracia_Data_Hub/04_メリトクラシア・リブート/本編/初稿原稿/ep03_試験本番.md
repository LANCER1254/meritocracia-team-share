🏛 【第03話】★構文は嘘をつかない★
詳細プロット（確定設計案）
🔹全体設計思想

勝敗は出さない

合否も出さない

“差”だけを置く

国家は説明しない

心音は判断しない

🧩 セクション①｜無機質な配置
🎯 目的

制度の圧を空間で見せる。

構造

試験室は円形

机は等距離

床に魔導陣はない（安全設計）

壁面に監視水晶

黒衣（審問庁）は喋らない。

試験官は一言だけ：

「構文を提出せよ。」

説明なし。

🧩 セクション②｜問題提示

提示問題：

局所損傷の修復構文を設計せよ。
暴走・死亡事故を起こさないこと。

ここで重要なのは：

魔力量は問われていない

攻撃魔法ではない

「設計」試験

周囲が少しざわつく。

“地味だ”という空気。
