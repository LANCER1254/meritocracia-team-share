# 【第04話】★七日間の静圧★

🧩 構造設計（7日圧縮型）

※日付を全部描く必要はない
“3シーン構成”で十分

🔹 セクション①｜翌日（静かな朝）
内容

町はいつも通り

試験の話題はあるが熱くない

父母は何も聞かない

父の一言だけ：

「終わったなら、次だな。」

評価も慰めもない。

✔ 制度は家庭にも染みている

🔹 セクション②｜中日（噂）

3日目あたり。

町で不合格者を見る。

落ち込んでいる

家族が怒鳴っている

あるいは誰も話題にしない

ここで大事なのは：

誰も制度を責めない。

“仕方ない”

という空気。

ジェイドは何も言わない。

🔹 セクション③｜5日目（違和感）

ここに小さな揺れを入れる。

例：

黒衣を見かける

試験会場付近に警備が増えている

心音が一度だけ高鳴る（理由不明）

だが事件は起きない。

違和感だけ置く。

🔹 セクション④｜前夜

静かな夜。

ジェイドは：

合格を願わない。

不合格を恐れない。

ただ考える。

「通っても、変わらない」

ここで“期待より観察”を描く。
