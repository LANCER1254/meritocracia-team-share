# 【第05話】★掲示板の向こう側★

🏛 第5話テーマ
「資格は行動の理由にはならない」

これは強い。

かなり強い。

だから構造でミスると一気に軽くなる。

ここを精密に肉付けする。

🧩 セクション①｜朝・掲示板へ向かう道
🎯 目的：祝祭ではなく“制度の朝”を描く
🔹描写設計

朝は静かすぎる

風が強い

家の戸が閉まる音が硬い

誰も目を合わせない

母：

「……いってらっしゃい」

父：

「確認してこい」

それ以上ない。

💡重要

父は「おめでとう」も「頑張れ」も言わない。

なぜならこの国では：

合格＝祝福ではなく“通過”

制度が家族にも染みている。

読者に気付かせる。

🧩 セクション②｜騒動（アイリス初登場）
🎯 目的：

ジェイドが“資格なしで動く”準備

🔹群衆描写

掲示板近く：

ざわつき

「どけよ」

「名前あったか？」

「……なかった」

祝祭ではない。

結果待ちの空気。

🔹騒動の中心

カール：
横柄、声が大きいが怒鳴り散らすほどではない。
「商品管理がなってない」とか冷たい言い方。

アイリス：
まだ従順。
反論しない。
ただ謝っている。

ここ重要：

ジェイドは合格をまだ知らない。

彼はまだ「平民」。

🧠 心音の扱い

鳴らない。

爆発しない。

ただ：

胸の奥が一瞬だけ、波打つ。

それだけ。

彼は立ち止まる。

飛び込まない。

観察する。

🧩 セクション③｜介入
🎯 ここが構造の核

ジェイドの台詞は短い。

長い正義語り禁止。

例：

「……それ、やりすぎじゃないか？」

だけでいい。

カールの返し

軽く見下す。

「お前、何様だ？」

ここでジェイドは：

「通りすがりだ」

まだ“合格者”ではない。

ユミナ介入

ここ重要。

ユミナはジェイドを助けない。

彼女はカールを制止する。

理由は：

「ここは発表前の広場です。秩序を保ちなさい」

制度の声。

感情ゼロ。

🧩 セクション④｜掲示板

騒動収束。

静か。

ジェイド、ようやく掲示板を見る。

名前を探す描写は短く。

演出：

「あった。」

それだけ。

歓声なし。

BGMなし。

自己完結なし。

🔥 超重要ポイント：

アイリスはまだ知らない。

ジェイドも言わない。

彼は“合格者”になる前に動いていた。

だから：

合格は行動の理由ではなかった。

テーマ成立。

🧩 セクション⑤｜視線

ここで物語を一段上げる。

🔹ユミナの視線

彼女は合格者として見ていない。

「あの少年は、合否前に動いた」

それを記録する。

🔹黒衣（審問庁）

掲示板ではなく、騒動を見ていた。

これで提示できる：

審問庁は試験だけ見ているわけではない。

行動も見ている。

🏛 この回の完成形

ジェイドは：

合格前に動く

合格後も何も変わらない

宣言もしない

誇らない

ただ帰る。

父が一言：

「あったか」

ジェイド：

「あった」

終わり。
