# 第メリトクラシアエピソードゼロ　堕天使邂逅−１話（初稿）

## 本文
シーン１
オレがその少年に会ったのは、六歳の頃だった。

場所は覚えていない。
ただ、本格的に夏が来る前で、
雨がひとしきり降った夜だったことだけは覚えている。

厚い雲に覆われ、月も見えなかった。

濡れた地面に、血が落ちていた。

雨で薄まっていたのに、
鉄みたいな匂いだけが、妙に強かった。

気がつくと、目の前に誰かがいた。

同じくらいの背丈。
服は破れていて、背中に――何か、あった。

破れた布の奥に、白いものが見えた気がする。

翼のようにも見えたけど、
はっきりとは思い出せない。

血が流れていた。
止まっていなかった。

「……大丈夫？」

気づいたら、そう声をかけていた。

理由は分からない。
怖かったはずなのに、体が先に動いた。

そのまま帰ることも、できたはずだった。

でも――

なぜか、足が止まらなかった。

手が震えて、うまくいかなかった。
包帯も、すぐに赤く染まった。

「……キミは、ボクが怖くないの？」

そう聞かれて、言葉に詰まった。

怖くないわけじゃなかった。

でも、目の前で倒れているのを見て、
見なかったことにはできなかった。

「……関係ないよ」

少年は少しだけ、困ったように笑った。

「ありがとう。でも、ここまでだ」

「これ以上、キミを巻き込みたくない」

立ち上がった瞬間、
空気が変わった気がした。

少年が、何かを口にした。

意味の分からない音だった。
言葉だったのかどうかも、覚えていない。

次の瞬間、
胸の奥が、強く鳴った。

《ドクン》

息が詰まって、視界が揺れた。

痛かったのかどうかも分からない。
ただ、心臓の音だけが、やけに大きく聞こえた。

「……大丈夫」

いつの間にか、名前を呼ばれていた。

どうして知っているのか、
聞く余裕はなかった。

「それじゃあ……」

少年が何か言いかけて、
その先は、思い出せない。

次に気づいた時、
そこには誰もいなかった。

雨は上がっていて、
地面には血の跡だけが残っていた。

夢だったのかもしれない。
そう思おうとした。

でも――

胸の奥だけが、ずっと落ち着かなかった。

理由は分からない。

ただ、その夜から、

オレは何かを“選ばされる”夢を、
よく見るようになった。
