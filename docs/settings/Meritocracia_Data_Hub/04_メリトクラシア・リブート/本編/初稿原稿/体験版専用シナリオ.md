# 第体験版専用シナリオ話（初稿）

## 本文
Scene1：教会（日常）

石床を滑る竹箒の音が、高い天井に反響して耳の奥をひっかいてくる。
　サッ、サッ、という乾いたリズム。
　
　窓から差し込む午後の光の中を、細かな埃がダンスを踊るように舞っている。

平和だな、と思う。

……退屈だ。

「ルカくん、こっちもお願いね」

声をかけてきた老婦人に、ボクは無言で会釈を返した。
　彼女は祭壇の前に供えられた花を丁寧に整え、満足そうに微笑んで去っていく。
　ボクは言われた場所へ歩き出し、また無心で手を動かした。

教会の中は、いつも通りの匂いがする。
　喉の奥に残る、甘ったるい百合の花の香りと、古くなった蝋燭が溶ける、脂の混じった独特の匂い。

……腹が減ったな。

中庭からは、追いかけっこをする子供たちの笑い声が聞こえてくる。
　何も考えずに笑っていられる、平坦な日常。
　
　サッ。

箒を動かす手が、ふと止まった。
　手のひらに伝わる振動が、ほんの少しだけ「ズレ」た気がした。
　
　視界の端。広場を横切った影。
　灰色の、薄汚れたローブを着た男だ。
　
　……まただ。

この一時間で、同じ男を三回見ている。
　一度目は門の前。二度目は西側の回廊。そして今、三度目。
　
　男の足音は聞こえない。
　市場の喧騒は石壁に遮られて遠い。

胸の奥が、ざわつく。

理由は、分からない。

トクン。

やけに、大きく聞こえた。

静寂が、急に重い膜のように体にまとわりついてくる。
　子供の声も、さっきの老婆の微笑みも、まるで作り物の舞台装置から流れる音源のように、現実味を失っていく。
　
「……気のせい、か」

自分に言い聞かせるように、小さく呟いた。
　その声は、礼拝堂の空気に消えた。
　
　ふと視線を向ける。
　地下へと続く重い鉄扉が、わずかに揺れたような気がした。
承知いたしました。メタ的な説明を排除し、ルカの「観察眼」と「直感」に根ざした一人称視点でScene2をリライトします。
Scene2：空腹と報酬

掃除を終えて教会の裏口を出ると、湿った風が鼻腔をくすぐった。
　どこかの屋台で肉を焼いている匂いだ。あるいは、広場のパン屋から流れてくる香りだろうか。

ギュル、と。
　情けない音が、静かな裏路地に響く。

胃の奥が、焼けるように熱い。
　ボクは無意識に、腹のあたりを強く押さえた。
　
　足元に、茶色の塊が落ちているのが見えた。
　……パンの耳だ。
　
　泥にまみれ、乾燥してひび割れている。
　ボクはそれを、じっと見つめた。
　
　一歩、足を踏み出す。
　手を伸ばそうとして、止まった。
　
　——惨めだな。
　
　そんな感情すら、今の空腹には贅沢すぎる気がした。
　ボクが再び指先を動かそうとした、その時。

「そんなものを拾う必要はない」

低い、落ち着いた声がした。
　
　顔を上げると、そこに男が立っていた。
　灰色の、薄汚れたローブ。
　掃除中、回廊の影に見え隠れしていた、あの男だ。
　
　ボクは反射的に一歩下がった。
　逃げるべきだ。直感がそう告げている。
　
　だが、男が差し出した手のひらを見て、ボクの足は止まった。
　
　夕闇の入り口で、それは鈍い銀色の光を放っていた。
　
「……銀貨？」
　
　ボクの口から、掠れた声が漏れる。
　一枚あれば、腹一杯の食事と、数日分の安心が買える。
　
「働きに見合う報酬だ」
　
　男の声には、感情がなかった。
　ただ事実を告げるような、機械的な響き。
　
「……ボクは、まだ何もしてない」
　
「これからしてもらう。難しいことじゃない。少し、荷物を運ぶのを手伝ってほしいだけだ」
　
　男はそれ以上何も言わず、ボクの目の前まで歩み寄ると、その銀貨をボクの手に握らせた。
　
　冷たい。
　なのに、肌に触れた瞬間、そこから熱が伝わってくるような錯覚を覚えた。
　
　ボクは、銀貨を握りしめた。
　手のひらの肉に、硬い縁が食い込む。

「……分かった」
　
　ボクの声は、自分でも驚くほど小さかった。
　
　足元に落ちていたパンの耳を、一度だけ振り返る。
　それはもう、ただのゴミにしか見えなかった。
Scene3：接触

銀貨をポケットに押し込む。布越しに伝わるその重みは、さっきまでの空腹を麻痺させるには十分すぎた。

「何をすればいい」

ボクの問いに、男は答えず、ただ顎で教会の裏口を示した。
　促されるまま、ボクは再び教会の敷地内へと足を踏み入れる。

向かったのは、礼拝堂ではない。
　建物の影に隠れるように配置された、備品搬入用の小さな勝手口だ。
　そこには、三つの木箱が置かれていた。

「これを、地下の奥まで運べ」

男が指差した箱には、見慣れた教会の紋章――星の導き――が刻印された封蝋がされている。
　ボクは箱の一つを抱え上げた。
　
　……重い。
　
　箱の隙間から、わずかに匂いが漏れている。
　教会の「百合と蝋」の匂いとは違う。もっとこう、鉄が錆びたような、あるいは雨上がりの土を掘り返したような――。

ふと視線を感じて、ボクは男を見た。
　灰色のフードの奥、暗がりに沈んだ瞳が、じっとボクの手元を観察している。
　
　ボクは黙って視線を外し、地下へ続く階段へ向かった。
　
「……いいか。中を見るな。置いたらすぐに戻ってこい」

背中越しに、男の低い声が響いた。
　その言葉は冷たい氷の粒のように、ボクの首筋を撫でていった。

一歩、階段を降りる。

一段ごとに、地上の喧騒が遠ざかっていく。
　
　ただ、運ぶ。
　
　それでいい。
　
　暗い階段の下から、生温かい風が吹き上がってきた。
　それは教会の地下室にあるはずのない、淀んだ熱を帯びていた。
Scene4：ペンダント（リライト）

地下への階段を数段降りたところで、背後から男に呼び止められた。

「待て。これも持っていけ」

振り返ると、男が指先で何かを摘んでいた。
　細い革紐の先に、親指ほどの小さな石がついたペンダントだ。

「……何、これ」

「お守りのようなものだ。運んでいる間、肌身離さず持っていなさい」

男はボクの返事を待たず、それをボクの首にかけた。
　
　ひやり、とした。
　石が鎖骨の間に触れた瞬間、心臓が跳ねた。

……温かい。
　
　いや、生温かい。
　石のくせに、まるで誰かの体温をそのまま移したような、不自然な熱を持っている。

トクン。

また、あの音だ。
　首元に触れている石が、ボクの鼓動に合わせて、わずかに震えているような気がした。
　
　ボクは指先で、その石に触れる。
　表面は滑らかだが、どこか粘りつくような質感がある。
　
　……気持ち悪い。
　
　指が、止まる。
　
　石を掴んだまま、動かない。
　
　喉の奥が、せり上がってくるような不快感。
　
　けれど、ポケットの中には、まだあの銀貨がある。
　硬くて、重い、救いの塊。

ボクは、石を握った手を離した。
　
「……分かった」

男は答えなかった。
　影の中に立つその姿は、もう壁の一部のようにしか見えない。
　
　ボクは首元の不気味な熱を無視して、再び階段を降り始めた。
　
　一歩、踏み出す。
　
　トクン。
　
　鼓動の音が、一段ごとに重くなっていく。
Scene5：地下への移動
階段を下りるボクの背後で、重い鉄の扉が動く音がした。

ギギッ、と錆びた蝶番が悲鳴を上げる。

バタン。

低い、腹に響くような音が地下通路に反響した。
　同時に、遠くで聞こえていた市場の喧騒も、中庭で遊ぶ子供たちの笑い声も、すべてが嘘みたいに消えた。
　
　残ったのは、ボクの引きずるような足音と、湿った沈黙だけだ。

……暗い。

壁に埋め込まれた燭台の火が、弱々しく揺れている。
　空気は地上よりも密度が高く、吸い込むたびに肺の奥が重くなるような感覚があった。
　
　一歩、進む。

カツン。

石床を叩く自分の足音が、やけに尖って聞こえる。
　首元の石が、また熱を持った。
　
　トクン。

鼓動が、耳のすぐ近くで鳴っている気がした。
　いや、これはボクの心臓じゃない。
　足元の箱から、あるいはこの壁の向こうから、別の何かが拍動しているような――。
　
　ボクは視線を足元に落としたまま、ただ前だけを見て歩いた。
　
　通路の脇に、古びたローブが積み上げられているのが見えた。
　その奥の壁には、見たこともない複雑な紋章が刻まれている。
　
　視線が、逸れる。
　
　視界の端に映る不穏な断片を、ボクは一つひとつ意識の外へ追い出していく。
　
　ただ、箱を運ぶ。
　
　通路の突き当たりに、さらに深い闇へと続く下り階段が見えた。
　そこから、さっきよりもいっそう生温かい風が、粘りつくように吹き上がってくる。
　
　ボクは止まらず、その闇の中へ足を踏み入れた。
Scene6：秘密（リライト）

階段を下りきった先は、広大な空洞になっていた。
　地上の礼拝堂の真下にあたる場所だろうか。天井は高く、湿った石壁が音を吸い込んでいる。

そこには、三人の男がいた。
　一人は、ボクに荷運びを頼んだあの灰色のローブ。
　あとの二人は、それよりも深い、闇のような黒い衣を纏っている。

男たちは、床に描かれた巨大な円を囲んでいた。
　
　紋章ではない。
　複雑に絡み合う線と、見たこともない歪な文字。
　その溝には、どろりとした赤黒い液体が流し込まれていた。
　
　鼻を突くのは、強烈な鉄の匂い。
　
　視線が、勝手に細部を拾う。
　
　黒い衣の男が持つ、白銀のナイフ。
　円の中心に置かれた、山積みの石。
　そして――。
　
　壁際に積み上げられた、大量の木箱。
　ボクが今抱えているものと同じ、教会の封蝋がされた箱だ。
　そのいくつかは蓋が開いており、中から覗いているのは、鈍く光る金属の塊。
　見たこともない、複雑な機構。
　
　視界が、ぐらりと揺れた。
　
　首元の石が、激しく拍動を始める。
　
　トクン、トクン、トクン。
　
　それはもうボクの心拍ではなく、地下室全体を揺らす巨大な鼓動のようだった。
　耳の奥で、誰かの囁きが聞こえる。
　祈りのような、呪いのような、低く響く声。

「……置け」

短い、拒絶を許さない声。
　灰色のローブの男が、こちらを見ずに言った。
　
　ボクは、動かない。
　
　ただ、箱を抱えたまま、足元の赤黒い模様を見つめている。
　
「置いたら、上に戻れ。見たことは忘れろ」
　
　男がゆっくりと振り返る。
　フードの奥に、冷たい光を宿した瞳が見えた。
　
　ボクは指先の感覚がないまま、抱えていた箱を床に置いた。
　
　ゴン、という鈍い音が、儀式場の静寂を切り裂いた。
　
　謝ることも、頷くこともせず、ボクは踵を返した。
　
　何も見なかったことにするように、視線を落とす。
　
　意識の端で、黒い衣の男がナイフを持ち上げるのが見えた。
　
　視線を逸らす。
　
　喉の奥までせり上がってきた鉄の匂いを飲み込み、ボクは暗い階段を駆け上がった。
Scene7：日常への回帰（リライト）

地下から這い上がるように階段を駆け上がり、鉄扉を押し開ける。
　
　西日が、目に刺さった。

「あ……」

思わず腕で顔を覆う。
　日差しが、やけに強い。
　夕暮れの街の、乾いた埃と、どこかの屋台が焼いている香草の匂い。
　
　ガチャン、と背後で扉が閉まる。
　
　ボクは教会の裏壁に背を預け、荒い呼吸を整えた。
　ポケットの中の銀貨に触れる。体温を吸った金属が、指先に張り付くようだった。

「――あ、ルカ！　見つけた！」

弾けるような声がした。
　
　顔を上げると、教会の角から一人の少女が駆け寄ってくるところだった。
　アンバーオレンジの髪をポニーテールにまとめ、白いエプロンを風になびかせている。
　
　カーネル姉ちゃんだ。

「もう、どこに行ってたの？　神父様がルカも上がっていいって言ってたよ。はい、これ！」

彼女はボクの目の前に、紙に包まれた何かを突き出した。
　
　焼きたてのパンの匂い。
　小麦の甘い、暴力的なまでに正しい「生の匂い」。
　
　鼻を突いていたあの鉄の臭みが、一瞬で上書きされる。

「……これ」

「売れ残りだけどね。今日もお疲れ様！」

カーネルは屈託のない笑みを浮かべて、ボクの顔を覗き込んだ。
　そのアンバーの瞳には、夕日の光が透き通るように反射している。
　
　口が開く。
　
　すぐに、閉じる。
　
　鉄の匂いも、喉までせり上がってきた何か。
　それをそのまま、飲み込む。

「……ありが、とう」

「どういたしまして！　あ、その首のやつ何？　新しいお守り？」

カーネルの指先が、ボクの首元のペンダントへ伸びる。
　
　動かない。
　
　ただ、彼女の指が不気味な熱を持った石に触れるのを、黙って見ていた。
Scene8：沈黙の代償（リライト）

ベンチに置かれた石は、夕闇の中でいっそう赤黒く淀んで見えた。
　ドクン、ドクン、と。
　木製の座面を伝って、不規則な震動がボクの指先にまで届く。

「わ、これ、なんかあったかい」

カーネル姉ちゃんが、躊躇いもなくその石を拾い上げた。
　彼女の指が触れた瞬間、ボクの首筋を焼いていた不快な熱が、嘘のように引いていく。
　
　肺に、空気が入る。

「……神父様に、持っていきなさいって、言われたんだ」

「へぇー！　いいなー、ルカばっかりズルい。ちょっと見せてよ」

彼女は石を目の高さに掲げ、透かすように夕日に向けた。
　アンバーの瞳に赤い光が混ざり、アンバーオレンジの髪がさらに鮮烈に、血のような色に染まる。
　
　口が開く。
　
　すぐに、閉じる。
　
　ボクは、目を逸らした。
　
「明日まで私が預かっておいてあげる。ルカ、さっきから顔色悪いし。ね、帰りに美味しいもの食べて元気出そう！」

彼女はこともなげに笑って、ペンダントを自分のエプロンのポケットに放り込んだ。
　
　肩が、軽い。
　
　ボクの首元には、もう何もない。
　
「……うん」
　
　ボクは小さく頷いた。
　
「決まり！　じゃあ行こ。お腹空いちゃった」
　
　カーネル姉ちゃんが先に立って歩き出す。
　その後ろ姿を、ボクは半歩遅れて追った。
　
　ポケットの中で石が脈打っているはずなのに、彼女の足取りはどこまでも軽やかだった。
　
　それでも、ボクは何も言わなかった。
