# 第堕天使の少年との邂逅−１話（初稿）

## 本文
Scene0　雨の夜

オレがその少年に会ったのは、六歳の頃だった。

場所は覚えていない。
ただ、本格的に夏が来る前で、
雨がひとしきり降った夜だったことだけは覚えている。

厚い雲に覆われ、月も見えなかった。

濡れた地面に、血が落ちていた。

雨で薄まっていたのに、
鉄みたいな匂いだけが、妙に強かった。

気がつくと、目の前に誰かがいた。

同じくらいの背丈。
服は破れていて、背中に――何か、あった。

破れた布の奥に、白いものが見えた気がする。

翼のようにも見えたけど、
はっきりとは思い出せない。

血が流れていた。
止まっていなかった。

「……大丈夫？」

気づいたら、そう声をかけていた。

理由は分からない。
怖かったはずなのに、体が先に動いた。

そのまま帰ることも、できたはずだった。

でも――

なぜか、足が止まらなかった。

手が震えて、うまくいかなかった。
包帯も、すぐに赤く染まった。

「……キミは、ボクが怖くないの？」

そう聞かれて、言葉に詰まった。

怖くないわけじゃなかった。

でも、目の前で倒れているのを見て、
見なかったことにはできなかった。

「……関係ないよ」

少年は少しだけ、困ったように笑った。

「ありがとう。でも、ここまでだ」

「これ以上、キミを巻き込みたくない」

立ち上がった瞬間、
空気が変わった気がした。

少年が、何かを口にした。

意味の分からない音だった。
言葉だったのかどうかも、覚えていない。

次の瞬間、
胸の奥が、強く鳴った。

《ドクン》

息が詰まって、視界が揺れた。

痛かったのかどうかも分からない。
ただ、心臓の音だけが、やけに大きく聞こえた。

「……大丈夫」

いつの間にか、名前を呼ばれていた。

どうして知っているのか、
聞く余裕はなかった。

「それじゃあ……」

少年が何か言いかけて、
その先は、思い出せない。

次に気づいた時、
そこには誰もいなかった。

雨は上がっていて、
地面には血の跡だけが残っていた。

夢だったのかもしれない。
そう思おうとした。

でも――

胸の奥だけが、ずっと落ち着かなかった。

理由は分からない。

ただ、その夜から、

オレは何かを“選ばされる”夢を、
よく見るようになった。



シーン１
ボクを助けてくれた幼い彼は、雨の中に消えていった。

小さな足音は、すぐに聞こえなくなった。

ボクは、しばらくその方向を眺めていた。

「……ふう」

息を吐く。

肩の傷から、まだ血が流れている。

人間の身体なら、とっくに動けなくなっている傷だ。

それでもボクは、壁にもたれたまま空を見上げた。

厚い雲が、夜空を覆っている。

月も星も見えない。

雨だけが、静かに落ち続けていた。

「面白い子だな」

小さく呟く。

人間は、基本的に臆病だ。

特に子供は、もっと分かりやすい。

血を見れば逃げる。
異様なものを見れば叫ぶ。

そういう生き物だ。

だからこそ――

「逃げなかった」

口元が、わずかに歪む。

怖い、と言った。

それでも、手を伸ばした。

理由は分からない、とも言った。

それでいい。

理由なんて、大した意味はない。

ボクは自分の胸に手を当てた。

鼓動は、もう落ち着いている。

けれど――

ほんのさっき、
あの子の胸が鳴った瞬間。

空気が、ほんの僅かだけ歪んだ気がした。

気のせいかもしれない。

それでもボクは、もう一度だけ、
彼が消えていった雨の向こうを見た。

「……さて」

空を見上げる。

厚い雲の奥。

その向こうにある気配を、ボクはもう感じていた。

「やっぱり、来るよね」

小さく笑う。

「本当に真面目だなぁ」

雨は、まだ止みそうにない。

そして――
この夜は、まだ終わらない。
雨は、まだ止みそうにない。

そして――
この夜は、まだ終わらない。

ボクはゆっくりと壁から体を離した。

肩の傷がじくりと痛む。
血も、まだ止まっていない。

普通の人間なら、ここで動けなくなるだろう。

けれど、ボクには大した問題じゃない。

それよりも――

「……早いな」

小さく呟く。

空気が、少しだけ変わった。

雨音が、わずかに遠くなる。

静かな夜だったはずの街外れに、
ほんのわずかな“重さ”が落ちてくる。

隠しているつもりなんだろう。

気配は薄い。
普通の人間なら、まず気づかない。

でも。

「バレてるよ」

ボクは肩をすくめた。

空を見上げる。

厚い雲の向こう。

「……あーあ」

ため息が漏れる。

「本当に真面目だなぁ」

追跡命令。

執行官。

天界の連中は、そういうのが好きだ。

規則。
秩序。
正義。

そういう言葉を並べて、
自分たちの行動を正当化する。

まあ、それはそれで嫌いじゃない。

分かりやすいから。

そのときだった。

雲の奥が、わずかに光る。

雷じゃない。

光は、音もなく真っ直ぐ落ちてくる。

まるで、空そのものが開いたみたいだった。

「……来た」

ボクは軽く首を鳴らす。

光はゆっくりと地面へ近づき、
雨粒を弾きながら降りてくる。

そして。

石畳の上に、静かに着地した。

雨が、わずかに揺れる。

そこに立っていたのは、一人の男だった。

長い外套。

背中には、大きな翼。

白い羽根が、雨の中で静かに揺れている。

男は、まっすぐボクを見た。

その目には、迷いがない。

そして、口を開く。

「……バアル様」

その呼び方に、ボクは思わず笑った。

「あー」

「その名前、まだ使ってるんだ」

男は動かない。

ただ、静かに言葉を続ける。

「天界法、第七条」

「堕天者の帰還命令を執行します」

雨が、二人の間に落ちる。

ボクは少しだけ首を傾けた。

「帰還命令？」

「それ、本気で言ってるの？」

男の目は、少しも揺れない。

「命令です」

「従わない場合、拘束。もしくは処刑」

ボクは肩をすくめた。

「相変わらずだなぁ」

「ほんと、融通きかないよね」

そして、少しだけ笑う。

「それにさ」

雨の向こう、
さっき少年が消えていった方向を見る。

「もう遅いよ」

視線を戻す。

目の前の天使を見て、ボクは言った。

「その名前は」

「さっき、雨の中に捨てたんだ」

男の目が、わずかに細くなる。

雨は、まだ降り続いている。

そして――

その夜の出来事は、
ここから静かに崩れ始める。

Scene2　断罪天使

雨は、静かに降り続いている。

白い翼が、雨粒を弾きながらゆっくりと揺れていた。

ボクの前に立つ天使――
エルディオンは、微動だにしない。

まるで、石像みたいだ。

「……バアル様」

もう一度、その名前を呼ぶ。

ボクは肩をすくめた。

「本当にそれ好きだよね、その呼び方」

「やめたんだけどな、ボク」

エルディオンの視線は変わらない。

「あなたは天界の秩序を破壊しました」

「堕天者として記録されています」

「よって――」

淡々と続ける。

「帰還命令を執行します」

ボクは少しだけ首を傾けた。

「帰還？」

「ボクが？」

軽く笑う。

「それ、本気？」

エルディオンは答えない。

ただ、静かにボクを見ている。

その目には、迷いがない。

昔から、そうだった。

真面目で、融通が利かなくて。

そして――

「相変わらずだなぁ」

ボクは小さく呟いた。

雨の中、少しだけ前に出る。

肩の傷から血が落ちる。

石畳に赤い滴が広がった。

エルディオンの視線が、その傷を一瞬だけ追う。

「人間を盾にするとは」

低い声が落ちた。

「……堕ちましたね」

ボクは思わず吹き出した。

「盾？」

「あはは」

肩を揺らして笑う。

「さっきの子？」

「違うよ」

軽く手を振る。

「勝手に助けてくれただけ」

エルディオンの眉が、わずかに動いた。

「……理解不能です」

「人間は、弱い存在です」

「利用する価値はあっても――」

「守る理由はない」

ボクは少しだけ目を細めた。

「守ったわけじゃないよ」

「面白かっただけ」

雨が、静かに降る。

しばらく沈黙が落ちる。

そしてエルディオンが、ゆっくりと右手を上げた。

その動きは、無駄がない。

「天界法に基づき」

静かな声。

「対象：堕天者バアル」

「拘束、または処刑を執行します」

空気が、わずかに震えた。

詠唱が始まる。

エルディオンの周囲に、淡い光が浮かぶ。

白い羽根が光を反射し、雨の中で輝いた。

「聖域展開」

石畳の上に、光の紋様が広がる。

「……お」

ボクは少しだけ感心した。

「いきなりそれ？」

「ずいぶん本気じゃない」

エルディオンは答えない。

ただ、詠唱を続ける。

「エクス――」

空の雲が、わずかに揺れた。

光が、集まり始める。

「カテドラ」

次の瞬間。

空から、巨大な光の槍が落ちてきた。

真っ直ぐに。

ボクの頭上へ。

雨を切り裂きながら――
それは、一直線に降りてくる。
