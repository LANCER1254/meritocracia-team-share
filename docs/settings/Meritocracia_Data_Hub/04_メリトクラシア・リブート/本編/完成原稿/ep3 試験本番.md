# 【第ep3 試験本番話】★タイトル★

🟦 Scene3-1｜講義室（決定稿）

扉を抜ける。
空気が、わずかに重くなった。

正面に、一段ずつの高さが揃った階段。
角は磨き抜かれ、影が直線を描いている。
横へ、机が並ぶ。
一直線。
ずれはない。
列と列の間隔も、同一の規格で切り取られている。

入口は、いちばん低い位置にある。
奥へ行くほど、床はせり上がっていく。
見上げる視界の最上段には、長い机。
逆光の中に、黒い人影が並んでいる。

番号を確認し、指定された席へ向かう。
椅子を引く音が、短く鳴った。
それきり、誰も喋らない。

机の右上に、認証盤。
光を持たない黒い円盤が、平滑に埋め込まれている。

隣の受験者とも、距離は正確だ。
近すぎず、遠すぎない。
互いの呼吸音すら、防音の壁に吸い込まれて届かない。
椅子は硬質な木製だ。
表面は滑らかに磨かれている。
軋みはない。

定められた型に、背を預ける。

背中に、上段からの視線が落ちてくる。


広さは、感じない。
ただ、　　
揃いすぎている。その事実だけが、静かに喉元を刺した。
