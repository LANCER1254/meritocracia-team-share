プロローグ・パート3（案）
――Scene③：母の謝罪

家に戻ったのは、夕方だった。

粉の袋は途中で拾い集めたが、
中身はもう、ほとんど残っていなかった。

扉を開けると、
母はすぐに異変に気づいた。

「……ジェイド？」

何も言わずに立ち尽くしていると、
視線が、オレの拳に落ちた。

赤く腫れていた。
指の関節が、少しだけ皮膚を破っている。

「……喧嘩？」

小さく、息を吸う音。

オレは答えなかった。
言葉にすると、
何かが壊れる気がした。

母は何も聞かずに、
桶に水を張った。

布を絞り、
オレの手を取る。

「痛む？」

首を横に振った。

本当は、少し痛かった。
でも、それより――
胸の奥が、変に静かだった。

水に触れた瞬間、
心臓が、わずかに脈打った。

《ドクン》

一瞬だけ。
音は、すぐに元に戻った。

母は気づいていない。

「……ジェイド。何があったの？」

今にも泣き出しそうな顔で、
母が、オレを見る。

「話せるならでいいから……教えて」

オレは、少しだけ間を置いてから、
昼間の出来事を、淡々と話した。

貴族と思われる少年たちに絡まれたこと。
名前を、笑われたこと。
両親のことを言われて、
腹が立って――手を出してしまったこと。

感情をつけると、
途中で止まりそうだったから。

事実だけを、順番に並べた。

全部話し終えると、
母は、何も言わずに立ち尽くしていた。

それから、
ゆっくりとオレを抱きしめた。

「……ごめんね」

声が、少し震えていた。

何に対する謝罪なのかは、
言われなくても分かった。

オレは、何も言わなかった。

その代わり、
母の服を、ぎゅっと掴んだ。

――名前負けしてんぞ。

「……違う」

自分でも驚くほど、
はっきり声が出た。

母の手が止まる。

「この名前が、悪いんじゃない」

言い切った瞬間、
胸の奥が、少しだけ重くなった。

「……オレが、弱いだけだ」

母は、何も言わなかった。

ただ、
布を持つ手が、わずかに震える。

「……ジェイド」

名前を呼ばれる。

それは、
昼間に聞いたものとは、違う音だった。

「あなたは、何も悪くない」

それだけ、言った。

理由も、説明もなかった。

オレは、黙って頷いた。

納得したわけじゃない。
でも――
分からないままにしておく、という選択はできた。

世界は、平等じゃない。

それだけは、はっきりしている。

「……オレ」

言いかけて、
言葉を飲み込んだ。

何を目指すのか。
どうなりたいのか。

まだ、うまく言えなかった。

母は、何も言わずに微笑った。

それは、
肯定でも、否定でもない顔だった。

夜になって、
布団に入ってからも、眠れなかった。

昼の出来事が、
何度も、断片みたいに浮かぶ。

名前。
拳。
笑い声。

そして、
銀色の髪の少年の声。

――悔しかったら、成り上がってこいよ。

心臓が、もう一度だけ鳴った。

《ドクン》

はっきり聞こえた。

でも、
意味は分からない。

分からないまま、
目を閉じた。

その夜、
久しぶりに夢を見た。

何かを選ばされる夢。

ただ――
目の前にあったのは、
道じゃなくて。

立ち止まれない場所だった。
