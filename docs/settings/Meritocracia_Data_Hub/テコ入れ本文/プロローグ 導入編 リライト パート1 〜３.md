プロローグ：導入編（リライト）
【パート1：六歳の邂逅】〈編集後〉
今になって思えば――
あの夜が、すべての始まりだったのかもしれない。

オレがその少年に会ったのは、六歳の頃だった。

場所は覚えていない。
ただ、本格的に夏が来る前で、雨がひとしきり降った夜だったことだけは覚えている。
曇天の空で、月も見えない。濡れた地面に、知らない血が落ちていた。

気がつくと、目の前に誰かがいた。
同じくらいの背丈。服はボロボロに破れていて、その背中には――翼のようにも見える「異形のもの」が張り付いていた。
はっきりとは思い出せない。ただ、そこから絶え間なく血が流れ、止まっていなかった。

「……大丈夫？」

怖かったはずなのに、身体が先に動いた。
震える手で包帯を巻こうとしたけれど、布はすぐに赤く染まっていく。

「……キミは、ボクが怖くないの？」

少年は力なく問いかけた。

「……関係ないよ」

そう答えるしかなかった。
目の前で消えそうな命を、見なかったことにはできなかった。

少年は少しだけ、困ったように笑った。

「ありがとう。でも、ここまでだ。これ以上、キミを巻き込みたくない」

立ち上がった瞬間、空気が凍りついた。
少年が何かを口にする。意味の分からない、けれど魂を直接揺さぶるような音。

次の瞬間、胸の奥が、物理的に押し潰されるほど強く鳴った。

《ドクン》

息が詰まり、視界が歪む。

「……大丈夫。いつか、自分で選べるようになる」

少年が、最後にそう言った気がした。

次に気づいた時、そこには誰もいなかった。
雨は上がり、地面には冷たくなった血の跡だけが残っていた。

その夜からだ。
オレは何かを“選ばされる”不気味な夢を、よく見るようになった。

【パート2：八歳の屈辱】〈編集後〉

あれから二年。
オレは八歳になり、なんとなく「こうすればいいんじゃないか」という直感が働くようになっていた。

母からお使いを頼まれ、市場の粉屋へ向かった時のことだ。

「はい、お使いでしょ？ 気をつけて帰りなさい」

粉屋のお姉さんから袋を受け取り、店を出たところで――背後から、ねっとりとした声が飛んできた。

「なあ、お前。それ、粉屋の袋だろ？」

振り返ると、同い年くらいの少年が三人立っていた。
仕立ての良い服、磨かれた靴。一目で「上の階級」だと分かる。

「……そうだけど」

「じゃあさ、お前――ウンフェーイグ（無資格者）だろ？ 貴族様に向かって、その態度はどうなんだよ」

一人が鼻で笑った。
悪意というより、それが当然の摂理だと言わんばかりの蔑み。

「お前らには関係ないだろ。失せろ」

苛立って立ち去ろうとしたオレに、一際高い笑い声が浴びせられた。

「平民風情の癖に、躾がなってないなあ！
お前、名前なんていうの？」

「……ジェイド・レオンハルト」

名前を口にした瞬間、場が静まり返った。

「は？ ジェイド？ レオンハルト？」
「なんだよそれ。貴族様みたいな名前じゃねえか！」

彼らは腹を抱えて笑い出した。

「ウンフェーイグの癖に、随分立派だな。親が夢見すぎなんだろ」
「身の程ってもんを教えてやれよ」

――その瞬間。

頭の中が、真っ白になった。

何かを考えたわけじゃない。
ただ、気づいたら拳を叩きつけていた。
鈍い音がして、一人の少年の顔が横に跳ねる。

「……っ、この野郎！」

一斉に殴りかかってくる三人。
腕を掴まれ、地面に押し付けられ、抱えていた袋が破けて白い粉が舞った。

（……負ける。こんなところで）

悔しさが込み上げた瞬間、冷徹な声が割って入った。

「……お前ら、その辺にしとけ。目立つ」

淡い銀色の髪をした少年が立っていた。
舎弟たちは渋々手を離す。

地面に膝をつくオレに、その少年――ライナルトは、感情の欠片もない視線を向けた。

「先に手を出したのは、お前だ。……悔しかったら」

彼は口元だけで、薄く笑った。

「貴族にでも、成り上がってこいよ。その名前に見合う場所まで」

それは挑発ですらなかった。
ただの「条件」の提示。

彼らが去った後、散らばった白い粉の中に、オレの震える拳だけが残された。

【パート3：母の謝罪と決意】〈編集後〉

家に戻ったのは、夕方だった。

「……ジェイド？」

母さんはすぐに、オレの傷だらけの拳に気づいた。
今にも泣き出しそうな顔で、桶に水を張る。

布を絞り、オレの手を拭う母さんの手のひらは冷たかった。

「何があったの……？ 教えて」

オレは、起きたことを淡々と話した。
名前を笑われたこと。両親まで侮辱されたこと。
だから、殴ったこと。

母さんはオレを強く抱きしめ、声を震わせた。

「……ごめんね。こんな名前にして、ごめんね……」

それは、親としての愛ゆえの謝罪だった。
けれど、今のオレには、その言葉が――
あいつらの嘲笑よりも、深く突き刺さった。

謝ってほしいわけじゃない。
慰めてほしいわけでもない。

「……違う」

母さんの腕を、拒絶するように解く。

「この名前が、悪いんじゃない。……オレが、弱いだけだ」

母さんは、驚いたように目を見開いた。
オレは、その瞳をまっすぐ見返す。

「母さんが謝る必要なんてない。
オレが、この名前を“本物”にするだけだ」

その瞬間、胸の奥で再び、あの音がした。

《ドクン》

六歳のあの日から眠っていた何かが、
オレの言葉に呼応して、目覚めた。

母さんは気づかない。
けれど、オレには確信があった。

あいつらを見返すためじゃない。
母さんに謝らせないために。

オレは、“選べる側”に行かなければならない。

世界は平等じゃない。
なら、その不平等なハシゴを――
自分の手で、登り切るまでだ。
