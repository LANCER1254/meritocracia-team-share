CHARACTER 07 / THE CONSUMED

FRAN

「削られ続けても、まだ消えていない感情の残骸」

Core Definition

「誰かに選ばれることでしか、自分の存在を証明できなかった少女」

強いわけでも、壊れているわけでもない。
ただ、消耗しきった果てに反応することを止めてしまった個人の象徴として存在する。

Basic Profile

State フェーズ① 設計ロック（意図的未完成）
Origin パリア（準奴隷階層）
Position 使用人階層 / 選ばれる側
Survival 「誰かの所有物であること」が唯一の生存戦略
Consumed Passive Under Construction Weakness Remains Recovery Route

State Definition

第02話 Scene2-5「灰色の扉前」の描写に基づき、現在状態を以下で固定する。

確定事項
  • 慣れによる無反応： 恐怖がないのではなく、恐怖に慣れすぎて足取りが一定になっている。
  • 壊れていないが、正常でもない： 感情は死んでいないが、表に出す回路が摩耗している。
  • 存在の退色： 低彩度、明度差の弱さ。瞳にだけ幽かな光が宿る。

Structure

背景構造と階層
  • パリア階層： 奴隷ではないが、自由でもない社会的最下層。
  • 主従関係への依存： 自由よりも、明確な主従関係の方に安心を感じてしまう。
  • キールとの接点： 教会時代、無意識のキールに助けられた記憶が「初めての救い」になっている。

Personality & Logic

表層 無表情、反応が薄い、指示に忠実、静か
内面 臆病、消耗、しかし完全には折れていない「まだ終わっていない」心
行動原理 自分で決めない、流れに乗る、選ばれることを待つ
NGライン 自発的な判断、冷静な戦略、強い意志による単独行動

Contrast Axis

vs ジェイド 「選ぶ側」と「選ばれる側」の対比
vs キール 「生き残る意志」と「生き残っているだけ」の対比
vs フィーネ 「判断する者」と「判断を委ねる者」の対比
vs アイリス 「先に救われた者」と「これから救われる者」の対比

Design Policy

執筆・設計方針
  • 「成長」ではなく「回復」： 強くなるのではなく、奪われた選択権を取り戻し、本来の自分に戻る過程を描く。
  • 感情爆発ではなく「静かな変化」： 劇的覚醒ではなく、少しずつ自分の意志で足を踏み出す変化を重視する。
  • 意図的保留： 覚醒トリガー、過去詳細、最終的立ち位置は本編で自然発生させる。

Summary

フランは、本作のテーマである「制度」に最も削られた側の人間である。
彼女が「自分で選ぶこと」を知る物語は、作品全体の救いとして機能する。
