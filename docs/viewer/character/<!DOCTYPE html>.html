MERITOCRACIA SPECIAL BLEND

Iris Earlgray

"壊れる直前で淹れられた、最も芳醇な一杯"

Character Analysis

主人のために完璧なアールグレイを淹れる、レオンハルト家の有能なメイド。その献身は、救われたことへの純粋な感謝に基づいている。

【生存罪悪感】
「再販価値」があったために自分だけが生かされたという事実は、彼女の献身を「義務」ではなく「贖罪」へと変えている。
【夜化因子の揺らぎ】
「忌み子」として家族に疎まれ、名前すら奪われていた時間は、彼女の魂を夜の深淵へと固定した。ジェイドの光が届かなければ、彼女は今も暗闇の澱の中に沈んでいた。
