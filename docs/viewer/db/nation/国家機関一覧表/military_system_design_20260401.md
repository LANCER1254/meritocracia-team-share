## 1. 本日の結論（重複統合後の正式コア）

本日の議論で確定した中核は、**三兵団を増やさず内部機能の分化で国家の厚みを出す**方針である。

基本思想は以下の3軸に集約される。

* **ZG = 秩序**
* **FO = 武力**
* **VA = 真実**

主人公ジェイドはこの3思想すべてと衝突・共鳴しながら成長する。

---

## 2. 三兵団の正式内部構造（統合確定）

内部構造は読者理解とゲームUIを両立するため、以下の共通階層で統一する。

```text
兵団
└ 局
   └ 連隊
      └ 班
         └ 個人
```

さらに各兵団の内部機能は、以下の**3層テンプレ**で設計する。

```text
象徴層
実務層
特殊層
```

この3層を全兵団の黄金テンプレとして採用。

## 3. 三兵団正式テンプレ（重複除去版）

