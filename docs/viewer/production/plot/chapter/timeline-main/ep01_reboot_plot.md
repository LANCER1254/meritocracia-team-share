Scene12｜物陰から見ていたやり取り

掲示板へ向かう人の流れから少し外れ、ジェイドは校舎横の石柱の陰へ足を寄せる。

本来なら、このまま自分の番号を確認して終わるはずだった。

だが、視界の先でカールと取引先の男がアイリスを挟むように立っている。

人垣から少し外れた場所。
周囲の受験者たちは結果に気を取られ、そちらへはあまり注意を向けていない。

ジェイドは石柱の陰から、そのやり取りを見てしまう。

「壊した分は別で働いてもらう」
「ちょうどこの方が引き取ってくださる」

カールの声は冷たい。
取引先の男は薄く笑い、アイリスを見ている。

逃げ場を探すように、アイリスの視線が揺れる。

「……いや……」

小さな拒絶。

男が一歩近づく。
アイリスはさらに後ろへ下がるが、背中が壁に触れて止まる。

「……助けて……」

その声は小さい。
けれど石柱の陰にいるジェイドには、はっきり届く。

足を止めたまま、ジェイドはすぐには動かない。

自分には関係ない。
まだ結果も見ていない。
ここで関われば面倒になる。

頭ではそう分かっている。

それでも、葬式の日に止まらなかった鼓動がまた胸の奥で鳴る。

空席の食卓。
市場の焼け跡。
灰色の扉。

いくつもの景色が一瞬だけ重なる。

次の瞬間、気づけば足が前へ出ていた。

ジェイドは石柱の陰から姿を現す。

「……それ、やりすぎじゃないか？」

その一言で、場の空気が止まる。
