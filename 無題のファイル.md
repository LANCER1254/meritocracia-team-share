    `about:preferences#general`

### ✅ 2. 環境変数で強制ソフトレンダリング

端末で起動時に：

`export MOZ_ENABLE_WAYLAND=0 export LIBGL_ALWAYS_SOFTWARE=1 firefox`

または

`LIBGL_ALWAYS_SOFTWARE=1 microsoft-edge`

→ 一時的にCPU描画モードで起動（遅くなるが安定）。

---

### ✅ 3. Firefox限定：WebRenderを切る

`about:config` を開いて以下を設定：

`gfx.webrender.all → false layers.acceleration.force-enabled → false`

再起動。

---

### ✅ 4. Chrome/Edge 系共通：起動オプション変更

`/usr/share/applications/microsoft-edge.desktop` を編集して  
Exec行を以下のように変更：

`Exec=/usr/bin/microsoft-edge-stable --disable-gpu --disable-gpu-compositing %U`

保存後、再起動。

---

### ✅ 5. Mesa/ドライバ更新（根本対処）

`sudo apt update sudo apt install --reinstall mesa-utils mesa-vulkan-drivers sudo apt upgrade reboot`

→ `glxinfo | grep "OpenGL renderer"` で正しくGPUが認識されているか確認。
