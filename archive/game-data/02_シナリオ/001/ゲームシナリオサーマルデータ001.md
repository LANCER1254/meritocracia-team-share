# EP01_01 : はじまりの階段
#=====================================================

[SCENE_INFO]
TITLE: ★はじまりの階段★
STAGE: 王都メリトクラシア・行政塔前
BGM: bgm_intro01.ogg（静かな序章）
TIME: 朝
CHARACTERS: ジェイド（主人公）, フラン（少女）, ライナルト（貴族少年）, 試験官たち

------------------------------------------------------

[NARRATION]
オレの名前はジェイド・レオンハルト。十歳だ。
今朝の光はやたら眩しいのに、胸の奥は鉛を流し込まれたみたいに重かった。

[DIALOGUE]
ジェイド：「なんでかって？　今日は《グローリアテスト》の日だから。」

[STAGE_DIRECTION]
カメラ：上方向パン → 行政塔を映す。
背景：王都・行政塔遠景。朝の光が塔の縁でちぎれ、影が街に落ちる。

[NARRATION]
合否ひとつで、階級も未来も、呼び方すら変わってしまう。
ここは実力主義国家――メリトクラシア。

[NARRATION]
生まれや家柄は“言い訳”。
力がすべてだ。

[THOUGHT]
ジェイド（心の声）：「オレは平民階級……ウンフェーイグ。意味は“役立たず”。」

[STAGE_DIRECTION]
BGM: 低く緊張感のある音へフェードイン。

[NARRATION]
王都のどこからでも見える高い塔。
朝日を背に突き立つその影が、街を覆っていた。

[DIALOGUE]
ジェイド：「……目を上げろ、迷子になるな。」

[NARRATION]
その合言葉を胸の中で繰り返すたびに、ほんの少しだけ前に進める気がした。

------------------------------------------------------
# SCENE 2：試験会場前
------------------------------------------------------

[STAGE_DIRECTION]
背景：行政塔付近・古い学舎前。
SE: 足音・風の音。

[NARRATION]
試験会場は塔の近くの古い学舎。
石段を上り、踊り場で一度深呼吸した。

[STAGE_DIRECTION]
エフェクト：息の白さ／空気の冷たさを強調。

[NARRATION]
石に手を当てると、夜の冷たさがまだ残っていた。

[EVENT]
背後から軽い足音。少女登場。

[CHARACTER_APPEAR]
フラン（灰色の紋章を胸に、伏し目がち）

[DIALOGUE]
ジェイド：「……フラン？」

[NARRATION]
彼女はぼさぼさの髪に擦り切れた袖。
胸元の灰色の紋章――準奴隷階級“パリア”の印だった。

[NARRATION]
落第した者、罪を背負った家族、保護を失った子。
名前が消え、番号で呼ばれる“箱”。

[THOUGHT]
ジェイド（心の声）：「番号で呼ばれるって……こういうことか。」

[STAGE_DIRECTION]
効果音：小さな心臓音、間の静寂。

[DIALOGUE]
少年（ライナルト）：「お前まで落ちるなよ。」

[CHARACTER_APPEAR]
ライナルト（白銀の髪・紫の瞳）

[DIALOGUE]
ライナルト：「パリアになったら、もう誰も名前なんか呼ばない。番号は覚えやすいけどな。」

[THOUGHT]
ジェイド（心の声）：「……ふざけんな。」

[NARRATION]
勝者と敗者。沈黙して歩くフランと、笑って嘲るライナルト。
オレはウンフェーイグ。だけど――

[DIALOGUE]
ジェイド：「絶対に、名前を手放さない。」

[STAGE_DIRECTION]
光：塔の影に一筋の朝光。  
BGM：高揚パートに移行。

------------------------------------------------------
# SCENE 3：試験開始
------------------------------------------------------

[STAGE_DIRECTION]
背景：試験会場内部。
SE: ドア開閉音 → 机の軋み。

[NARRATION]
灰色の扉を押し開けると、冷たい空気が流れ込んだ。
紙とインクの匂い。机がびっしり並ぶ。

[THOUGHT]
ジェイド：「落ち着け。目を上げろ、迷子になるな。」

[EVENT]
黒衣の試験官が出現。目線がジェイドに止まる。

[STAGE_DIRECTION]
BGM: 無音 → 緊張の効果音。

[NARRATION]
基礎学力、魔力量、倫理――全てが“ふるい”。

[MINIGAME]
TITLE: グローリアテスト（模擬）
TYPE: 選択式対話
CHOICE:
  1. 落ち着いて解く（成功）
  2. 焦って書き間違える（失敗）
  3. 黒衣を気にして集中できない（ルート分岐）

[RESULT_1]
成功時：画面に白光。試験官が一瞬こちらを見る。

[RESULT_2]
失敗時：紙に滲む汗、視線が刺さる演出。

------------------------------------------------------
# SCENE 4：倫理試問（選択肢分岐）
------------------------------------------------------

[STAGE_DIRECTION]
背景：個室。
SE: ペンの音、時計の針音。

[NARRATION]
最後の試験は、倫理の問答だった。

[DIALOGUE]
試験官：「飢えた子を救うために、罪を犯すことは正しいか？」

[CHOICE]
1. 「正しい。誰かを救うためなら。」
2. 「正しくない。でも、見捨てたくない。」
3. 「答えられない。」

[RESULT_1]
ジェイド：「正しいかどうかなんて、オレにはわかんない。でも……見捨てるより、叱られる方を選ぶ。」

[RESULT_2]
ジェイド：「誰かを切り捨てるなんて……オレには耐えられない。」

[STAGE_DIRECTION]
BGM: 静寂。カメラ：試験官の無表情にズーム。

[NARRATION]
黒衣の視線が、また刺さった。

------------------------------------------------------
# SCENE 5：夕暮れの塔前
------------------------------------------------------

[STAGE_DIRECTION]
背景：夕暮れ・行政塔。  
BGM：エンディングテーマ "steps_of_merit.ogg"

[NARRATION]
日が傾き、塔の影が街を覆っていく。
今日、オレは“選ばれなかった”。でも――

[THOUGHT]
ジェイド（心の声）：「選ばれるのを待つだけじゃ、一生番号に続く階段だ。」

[DIALOGUE]
ジェイド：「だから、登る。自分で。」

[STAGE_DIRECTION]
カメラ：塔を見上げる。光が瞳に反射。  
SE: 紙がめくられる音。黒い袖が引くカットイン。

[NARRATION]
――誰かが、見ていた。
オレが名前を失わないかどうか。その瞬間まで。

[END_SCENE]
ジェイド：「ジェイド・レオンハルト。番号じゃない。これが、オレの最初の誓いだ。」

[FADE_OUT]
