# Meritocracia Team Share

本リポジトリは『メリトクラシア』制作チーム向けの共有資料置き場です。

## 構成
- docs/
  - prologue/    : プロローグ原稿（未着手）
  - settings/    : 世界観・制度設定
  - characters/  : キャラクター設定
  - meetings/    : 制作会議議事録
- assets/
  - images/      : 参考画像・資料
  - references/  : 外部参照資料
- archive/       : 過去データ・スナップショット

## 運用方針
- GitHubを正とする
- 大きな変更はcommitメッセージで明示
- 途中稿はarchiveへ退避
